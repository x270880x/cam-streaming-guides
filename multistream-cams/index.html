Multiple Cam Sites

How to broadcast to multiple cam sites at once

Free SplitCam can broadcast one encode to several cam sites at the same time — BongaCams, Chaturbate, CAM4, Stripchat and more. No watermark.

Quick answer
To broadcast to several cam sites at once: install SplitCam, build your scene, get the RTMP server URL and stream key from each cam site, add them all in SplitCam's multistream settings, click Go Live once.
  1. Install SplitCam.
  2. Add camera + scene.
  3. Get an RTMP key from each cam site.
  4. Add all keys in SplitCam multistream.
  5. Go Live once.

Step-by-step

1
Install SplitCam

SplitCam is free live-streaming software for Windows and macOS with built-in multistreaming. Download and install it — no watermark, no signup.

2
Set up your camera and scene

Open SplitCam, add your webcam and build your scene with overlays and filters. One scene feeds every destination.

3
Get an RTMP key from each cam site

On each cam platform, enable external / RTMP broadcasting and copy its server URL and stream key. Repeat for every site you want to broadcast to.

4
Add every destination in SplitCam

Open Stream Settings and add each cam site as a custom RTMP destination — paste its server URL and key. Tick all the ones you want live.

5
Click Go Live once

Press Go Live. SplitCam sends your stream to every selected cam site simultaneously, peer-to-peer, from one encode — no extra fee.

Pro tips

Mind your upload

Multistreaming multiplies upload load. Each destination needs its own bitrate — make sure your connection can carry the total.

Wired connection

Ethernet is essential when pushing several streams at once.

Check each platform's rules

Some cam sites restrict simultaneous broadcasting elsewhere — confirm before you multistream.

Watch the health monitor

SplitCam shows per-destination status — drop a site if your upload can't keep up.

FAQ

Is multistreaming free in SplitCam?

Yes — multistreaming is built in and free, no per-destination fee, no watermark.

How many cam sites can I broadcast to at once?

As many as your upload bandwidth supports — each destination consumes its own bitrate.

Does this use a cloud relay?

No — SplitCam sends streams peer-to-peer directly from your PC to each platform's ingest server.

Will multistreaming slow my PC?

Encoding is done once and reused; hardware encoding keeps CPU load low. Upload bandwidth is the real limit.
