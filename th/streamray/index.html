วิธีถ่ายทอดบน StreamRay ด้วย SplitCam

StreamRay มีการตั้งค่า encoder ภายนอกที่ไม่ธรรมดา — จัดส่ง URL สตรีมใน หน้าต่างแชทของการถ่ายทอด และไม่ใช้คีย์สตรีมแยกต่างหาก SplitCam ฟรีจัดการกระแสเฉพาะ URL นี้

คู่มือวิดีโอ

ทีละขั้นตอน

1
ดาวน์โหลดและติดตั้ง SplitCam

SplitCam คือซอฟต์แวร์สตรีมมิ่งสดฟรีสำหรับ Windows และ macOS ดาวน์โหลดและรันตัวติดตั้ง — ไม่ต้องสมัคร ไม่ต้องใช้บัตร ไม่มีลายน้ำ ไม่มีขีดจำกัดเวลา SplitCam คือ encoder ที่ส่งวิดีโอของคุณไปยัง StreamRay

2
ตั้งค่ากล้องและฉาก

เปิด SplitCam และเพิ่มเว็บแคม สร้างฉากตามที่คุณต้องการให้ผู้ชมเห็น — โอเวอร์เลย์ ข้อความ กล้องที่สองหรือโทรศัพท์ ฟิลเตอร์ความงามหรือพื้นหลัง AI ทุกอย่างจะถูกนำมาใช้แบบสดก่อนที่สตรีมจะออกจากพีซีของคุณ

3
รับสตรีมคีย์ StreamRay ของคุณ

บน StreamRay ดับเบิลคลิก Broadcast Now เปิดเมนู Other เลือก OBS Broadcaster และ Save and Close StreamRay จะโพสต์ URL สตรีมในหน้าต่างแชท — คัดลอกจากที่นั่น ปล่อยช่องคีย์สตรีมของ SplitCam ว่าง StreamRay พิสูจน์ตัวตนผ่าน URL เท่านั้น

4
เชื่อมต่อ SplitCam กับ StreamRay

ใน SplitCam เปิด Stream Settings วาง URL เซิร์ฟเวอร์ StreamRay และสตรีมคีย์ในช่อง RTMP กำหนดเอง บิตเรต: 3,500–6,000 Kbps สำหรับ 1080p, 2,000–4,000 Kbps สำหรับ 720p เรียกใช้การทดสอบความเร็วในตัวก่อน

5
คลิก Go Live

กด Go Live ใน SplitCam จากนั้นเริ่มการถ่ายทอดสดบน StreamRay ภายในประมาณ 10 วินาที กล้องของคุณก็จะออนไลน์ การสตรีมครั้งต่อๆ ไปทำได้ด้วยคลิกเดียว — เปิด SplitCam, Go Live

เคล็ดลับ

URL อยู่ในแชท

StreamRay ไม่แสดง URL สตรีมในกล่องการตั้งค่า — โพสต์ในหน้าต่างแชทของการถ่ายทอด คัดลอกจากที่นั่น

ปล่อยช่องคีย์ว่าง

StreamRay ไม่ใช้คีย์แยกต่างหาก — เฉพาะ URL อย่าใส่อะไรในช่องคีย์ของ SplitCam

ใช้การเชื่อมต่อแบบมีสาย

อีเทอร์เน็ตชนะ Wi-Fi ในการสตรีมที่ยาว — เฟรมที่หายคือทิปที่หาย ลากสายไปยังพีซีสตรีม

ทำการทดสอบส่วนตัวก่อน

รันการถ่ายทอดทดสอบสั้นๆ เพื่อตรวจสอบกล้อง เสียง การจัดเฟรม และโอเวอร์เลย์ก่อนเปิดห้องให้ผู้ชม

คำถามที่พบบ่อย

URL สตรีม StreamRay อยู่ที่ไหน?

หลังจากเปิดใช้งาน OBS Broadcaster StreamRay โพสต์ URL ในหน้าต่างแชท — คัดลอกจากแชท

ทำไมไม่มีคีย์สตรีมบน StreamRay?

StreamRay พิสูจน์ตัวตนผ่าน URL เท่านั้น — ปล่อยช่องคีย์ของ SplitCam ว่าง

บิตเรตสำหรับ StreamRay?

StreamRay ไม่ประกาศเพดานทางการ — เล็ง 3,500–6,000 Kbps ที่ 1080p และรันการทดสอบความเร็วของ SplitCam เนื่องจากกระแสเฉพาะ URL ไม่ให้ข้อเสนอแนะบิตเรต

SplitCam ฟรีสำหรับ StreamRay หรือไม่?

ใช่ — ฟรี ไม่มีลายน้ำและไม่จำกัดเวลา ซึ่งตรงกับกระแสเฉพาะ URL ของ StreamRay: วางลิงก์แล้วคุณก็ไลฟ์
