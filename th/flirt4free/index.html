Flirt4Free เป็นหนึ่งในแพลตฟอร์ม cam ที่เก่าแก่ที่สุด ออกอากาศตั้งแต่ทศวรรษ 90 รองรับการถ่ายทอดภายนอกอย่างเป็นทางการผ่าน External Broadcast Form — SplitCam ฟรีส่งสตรีมในขณะที่ฟอร์มตั้งค่าความละเอียดและบิตเรต

คำตอบด่วน
ถ่ายทอดบน Flirt4Free ด้วย SplitCam: ติดตั้ง SplitCam สร้างฉาก เปิด Flirt4Free External Broadcast Form คัดลอก RTMP URL และ Stream Name และตั้งความละเอียด/บิตเรต วางใน SplitCam Go Live

คู่มือวิดีโอ

ทีละขั้นตอน

1
ดาวน์โหลดและติดตั้ง SplitCam

SplitCam คือซอฟต์แวร์สตรีมมิ่งสดฟรีสำหรับ Windows และ macOS ดาวน์โหลดและรันตัวติดตั้ง — ไม่ต้องสมัคร ไม่ต้องใช้บัตร ไม่มีลายน้ำ ไม่มีขีดจำกัดเวลา SplitCam คือ encoder ที่ส่งวิดีโอของคุณไปยัง Flirt4Free

2
ตั้งค่ากล้องและฉาก

เปิด SplitCam และเพิ่มเว็บแคม สร้างฉากตามที่คุณต้องการให้ผู้ชมเห็น — โอเวอร์เลย์ ข้อความ กล้องที่สองหรือโทรศัพท์ ฟิลเตอร์ความงามหรือพื้นหลัง AI ทุกอย่างจะถูกนำมาใช้แบบสดก่อนที่สตรีมจะออกจากพีซีของคุณ

3
รับสตรีมคีย์ Flirt4Free ของคุณ

ในพื้นที่โมเดลของ Flirt4Free เปิด External Broadcast Form ไม่เหมือนไซต์ cam ส่วนใหญ่ Flirt4Free ให้ RTMP URL และ Stream Name (ไม่ใช่ «คีย์») รวมถึงฟิลด์ความละเอียดและบิตเรตที่คุณกรอกในฟอร์มเอง คัดลอก URL และ Stream Name ในช่องเซิร์ฟเวอร์และคีย์ของ SplitCam

4
เชื่อมต่อ SplitCam กับ Flirt4Free

ใน SplitCam เปิด Stream Settings วาง URL เซิร์ฟเวอร์ Flirt4Free และสตรีมคีย์ในช่อง RTMP กำหนดเอง บิตเรต: 3,500–6,000 Kbps สำหรับ 1080p, 2,000–4,000 Kbps สำหรับ 720p เรียกใช้การทดสอบความเร็วในตัวก่อน

5
คลิก Go Live

กด Go Live ใน SplitCam จากนั้นเริ่มการถ่ายทอดสดบน Flirt4Free ภายในประมาณ 10 วินาที กล้องของคุณก็จะออนไลน์ การสตรีมครั้งต่อๆ ไปทำได้ด้วยคลิกเดียว — เปิด SplitCam, Go Live

เคล็ดลับ

เป็น Stream Name ไม่ใช่คีย์

Flirt4Free เรียกหนังสือรับรองว่า Stream Name คุณวางในช่องคีย์สตรีมของ SplitCam — บทบาทเดียวกัน

ตั้งความละเอียด/บิตเรตในฟอร์ม

External Broadcast Form ของ Flirt4Free มีฟิลด์ความละเอียดและบิตเรตของตัวเอง — จัดให้ตรงกับการตั้งค่าของ SplitCam เพื่อไม่ให้ภาพถูกปรับขนาด

แพลตฟอร์มประวัติศาสตร์

Flirt4Free ทำงานตั้งแต่ทศวรรษ 90 — เครื่องมือโมเดลพัฒนาแล้ว และ External Broadcast Form เป็นส่วนหนึ่งที่บันทึกไว้

ใช้การเชื่อมต่อแบบมีสาย

อีเทอร์เน็ตชนะ Wi-Fi ในการสตรีมที่ยาว — เฟรมที่หายคือทิปที่หาย ลากสายไปยังพีซีสตรีม

คำถามที่พบบ่อย

Flirt4Free รองรับ encoder ภายนอกหรือไม่?

ใช่ — อย่างเป็นทางการ ผ่าน External Broadcast Form ที่ให้ RTMP URL และ Stream Name SplitCam ทำงานเป็น encoder

ฉันรับข้อมูล RTMP ของ Flirt4Free ที่ไหน?

จาก External Broadcast Form ในพื้นที่โมเดล — แสดง RTMP URL, Stream Name และฟิลด์ความละเอียด/บิตเรต

บิตเรตสำหรับ Flirt4Free?

3,500–6,000 Kbps ที่ 1080p — ตั้งค่าเดียวกันใน External Broadcast Form และใน SplitCam

SplitCam ฟรีสำหรับ Flirt4Free หรือไม่?

ใช่ — ฟรี ไม่มีลายน้ำและไม่จำกัดเวลา ซึ่งเข้ากันกับแพลตฟอร์มประวัติศาสตร์อย่าง Flirt4Free ที่โชว์อาจยาวนาน
