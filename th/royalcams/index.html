วิธีถ่ายทอดบน RoyalCams ด้วย SplitCam

RoyalCams เป็นไซต์ cam ฟรีแบบ token — ห้องสาธารณะเปิดที่หล่อเลี้ยงด้วยทิป พร้อมโชว์ส่วนตัวเสริม การตั้งค่า broadcast รองรับเส้นทาง ตัวเข้ารหัสภายนอก มาตรฐานที่ SplitCam ฟรีเข้ามาเชื่อมต่อ ให้คุณสตรีมด้วยซีนหลายกล้อง โอเวอร์เลย์ และฟิลเตอร์ความงามแทนเว็บแคมเดียวแบบเรียบๆ

คำตอบด่วน
ถ่ายทอดบน RoyalCams ด้วย SplitCam: ติดตั้ง SplitCam จัดซีน ในแดชบอร์ดโมเดลเปิด broadcast settings → ตัวเข้ารหัสภายนอก คัดลอก server URL และ stream key วางใน SplitCam Go Live

ทีละขั้นตอน

1
ดาวน์โหลดและติดตั้ง SplitCam

SplitCam เป็นซอฟต์แวร์ถ่ายทอดสดฟรีสำหรับ Windows และ macOS — ไม่ต้องสมัคร ไม่ต้องใช้บัตร ไม่มีลายน้ำ

2
จัดซีน

เปิด SplitCam แล้วเพิ่มเว็บแคม ทับโอเวอร์เลย์ ข้อความเป้าหมายทิป กล้องที่สองหรือมือถือ ฟิลเตอร์ความงาม หรือพื้นหลัง AI

3
เอา URL และ stream key ของ RoyalCams

ล็อกอินบัญชีโมเดล RoyalCams เปิด broadcast settings → ตัวเข้ารหัสภายนอก คัดลอก server URL และ stream key

4
เชื่อม SplitCam เข้ากับ RoyalCams

เปิด Stream Settings ใน SplitCam วาง server URL และ stream key ของ RoyalCams ในช่อง RTMP กำหนดเอง ตั้งบิตเรต 3,500–6,000 Kbps ที่ 1920×1080, 30 fps, keyframe 2 วินาที

5
คลิก Go Live

กด Go Live ใน SplitCam แล้วเปลี่ยนสถานะเป็นออนไลน์จากแดชบอร์ดของคุณ ภายในประมาณ 10 วินาทีสัญญาณไปถึงผู้ชมของ RoyalCams

เคล็ดลับ

ห้อง token ตอบแทนคุณภาพงาน

ห้องสาธารณะของ RoyalCams ขับเคลื่อนด้วยทิป — โอเวอร์เลย์เป้าหมายทิปและซีนที่เนี้ยบเปลี่ยนคนแอบดูเป็นผู้ให้ทิปและโชว์ส่วนตัว

เปลี่ยนเป็นโชว์ส่วนตัว

ใช้ซีนสาธารณะที่แข็งแรงเพื่อขายต่อยอดโชว์ส่วนตัว ซึ่งเป็นที่ที่รายได้จริงอยู่บนไซต์ cam แบบ token

ตัวเข้ารหัสภายนอกปลดล็อกซีน

การส่งผ่าน RTMP ของ SplitCam แทนกล้องเบราว์เซอร์เปิดเลย์เอาต์หลายกล้อง โอเวอร์เลย์ และฟิลเตอร์

ใช้การเชื่อมต่อแบบมีสาย

อีเทอร์เน็ตชนะ Wi-Fi ในการสตรีมที่ยาว — เฟรมที่หายคือทิปที่หาย ลากสายไปยังพีซีสตรีม

การแก้ไขปัญหา

สตรีม RoyalCams ของคุณกระตุกหรือบัฟเฟอร์

เกือบทุกครั้งบิตเรตสูงกว่าที่อัปโหลดของคุณรับไหว รันการทดสอบความเร็วในตัวของ SplitCam แล้วตั้งบิตเรตที่ประมาณ 75% ของอัปโหลดที่วัดได้ — 3,500–6,000 Kbps สำหรับ 1080p ต่ำกว่าสำหรับ 720p อาการกระตุกจะหายไปเมื่อ encoder หยุดวิ่งแซงการเชื่อมต่อของคุณ

เฟรมหลุดระหว่างการถ่ายทอดบน RoyalCams

เฟรมหลุดหมายความว่าแพ็กเก็ตไปไม่ถึง RoyalCams ทันเวลา — มักเป็น Wi-Fi ไม่เสถียร เปลี่ยนเป็นการเชื่อมต่อ Ethernet แบบมีสาย ปิดแอปที่กินแบนด์วิดท์ และลดบิตเรตลงเล็กน้อย พีคครั้งเดียวไม่เป็นไร แต่การเพิ่มขึ้นต่อเนื่องหมายความว่าการเชื่อมต่อตามไม่ทัน

จอดำ — ผู้ชมไม่เห็นวิดีโอบน RoyalCams

กล้องของคุณไม่ได้ถูกเลือกเป็นแหล่งที่ใช้งานใน SplitCam หรือมีแอปอื่นยึดไว้ ปิด Zoom, Skype หรือ OBS เลือกเว็บแคมใหม่ในรายการแหล่งของ SplitCam และยืนยันว่าตัวอย่างแสดงภาพของคุณก่อนกด Go Live

RoyalCams ปฏิเสธสตรีมคีย์หรือไม่เชื่อมต่อ

คัดลอกสตรีมคีย์ใหม่ — ช่องว่างที่ท้ายหรือคีย์เก่าที่หมุนเปลี่ยนแล้วเป็นสาเหตุปกติ ยืนยันว่า URL เซิร์ฟเวอร์ตรงกับที่ RoyalCams แสดง และเปิดใช้งานการถ่ายทอดผ่าน encoder ภายนอกในบัญชีของคุณแล้ว แถบเลื่อนสีเขียวใน Stream Settings ของ SplitCam ยืนยันว่าคีย์ถูกต้อง

ไม่มีเสียงหรือเสียงไม่ตรงกับภาพบน RoyalCams

เลือก SplitCam เป็นทั้งกล้องและไมโครโฟน และเลือกไมโครโฟนจริงของคุณในแหล่งเสียงของ SplitCam หากเสียงตามหลังวิดีโอ ลดความละเอียดลงหนึ่งระดับ — encoder ทำงานหนักเกินไปและเสียงกำลังรอเฟรมที่มาช้า

คำถามที่พบบ่อย

RoyalCams รองรับตัวเข้ารหัสภายนอกอย่าง SplitCam ไหม?

รองรับ — การตั้งค่า broadcast เปิดเส้นทางตัวเข้ารหัสภายนอก / RTMP มาตรฐาน คัดลอก server URL และ stream key ลงใน SplitCam หลังการยืนยันตัวตน

เอา RoyalCams stream key จากที่ไหน?

ในการตั้งค่า broadcast / ตัวเข้ารหัสภายนอกของแดชบอร์ดโมเดล — ทั้ง server URL และ stream key อยู่ที่นั่น วางทั้งคู่ลงในช่อง RTMP กำหนดเองของ SplitCam

ควรใช้บิตเรตเท่าไรสำหรับ RoyalCams?

1920×1080 ที่ 30 fps, 3,500–6,000 Kbps, keyframe 2 วินาที รันการทดสอบความเร็วของ SplitCam ก่อน

SplitCam ฟรีสำหรับ RoyalCams ไหม?

ใช่ — SplitCam ฟรี ไม่มีลายน้ำและไม่จำกัดเวลา ตัวเลือกตัวเข้ารหัสภายนอกของ RoyalCams ในแดชบอร์ดก็เปิดให้ฟรี

โมเดลสามารถสร้างรายได้บน RoyalCams ได้เท่าไหร่?

รายได้บน RoyalCams ขึ้นอยู่กับขนาดของผู้ชม จำนวนชั่วโมงที่สตรีม และพฤติกรรมการให้ทิป ผู้แพร่ภาพที่ออนไลน์สม่ำเสมอมักจะได้รับ $200–$3,000 ต่อเดือน ส่วนผู้ที่อยู่ระดับท็อปทำได้ มากกว่า $10,000 ส่วนแบ่งรายได้ของคุณเป็นไปตามโครงสร้างค่าคอมมิชชันของ RoyalCams — ตรวจสอบสัญญาโมเดลก่อนเริ่มไลฟ์

RoyalCams ปลอดภัยสำหรับผู้แพร่ภาพหรือไม่?

RoyalCams กำหนดให้ยืนยันอายุและบัตรประชาชนก่อนรับเงิน ซึ่งช่วยปกป้องโมเดลจาก การฉ้อโกง ใช้ชื่อบนเวที อย่าเปิดเผยข้อมูลส่วนตัวหน้ากล้อง เปิดใช้งานการบล็อกตามภูมิภาคเพื่อ ซ่อนสตรีมของคุณจากพื้นที่บ้านเกิด และจัดการคำขอของผู้ชมทุกครั้งเป็นเชิงธุรกรรม โอเวอร์เลย์และพื้นหลัง AI ของ SplitCam ยังช่วยซ่อนหรือแทนที่สภาพแวดล้อมจริง ของคุณได้ด้วย

ต้องใช้เอกสารอะไรบ้างเพื่อเป็นโมเดลบน RoyalCams?

RoyalCams มักจะต้องใช้บัตรประจำตัวที่มีรูปถ่ายซึ่งออกโดยรัฐ (พาสปอร์ต ใบขับขี่ หรือบัตรประชาชน), เซลฟี่พร้อมถือบัตร และแบบฟอร์มภาษี/การจ่ายเงิน (W-9 สำหรับสหรัฐฯ, W-8BEN สำหรับนอกสหรัฐฯ) การอนุมัติมักใช้เวลา 24–72 ชั่วโมง เมื่อได้รับอนุมัติแล้ว คุณสามารถออนไลน์ได้ในวันเดียวกัน

ฉันสามารถสตรีมบน RoyalCams จากโทรศัพท์ได้หรือไม่?

โดยปกติ RoyalCams จะมีแอปผู้แพร่ภาพบนมือถือหรือผู้แพร่ภาพแบบ mobile-web แต่ ประสบการณ์ใช้งานจำกัด — ไม่มีโอเวอร์เลย์ ไม่มีกล้องตัวที่สอง ไม่มีพื้นหลัง AI สำหรับคุณภาพ การผลิตเต็มรูปแบบ ให้แพร่ภาพจากคอมพิวเตอร์ด้วย SplitCam และใช้โทรศัพท์เป็น กล้องตัวที่สอง (SplitCam รองรับอินพุตจาก IP-camera ของโทรศัพท์)

RoyalCams รองรับ OBS หรือ encoder ภายนอกหรือไม่?

รองรับ — RoyalCams จะให้ RTMP server URL และ stream key ในแผงผู้แพร่ภาพ วางทั้งสองค่าใน Stream Settings → Custom RTMP ของ SplitCam ตั้งค่า 1920×1080 ที่ 30 fps พร้อม bitrate 4,000–5,000 Kbps แล้วคลิก Go Live เส้นทาง Custom RTMP จะให้คุณใช้งานการจัดฉากเต็มรูปแบบของ SplitCam (กล้องหลายตัว, โอเวอร์เลย์, ฟิลเตอร์)
