วิธีถ่ายทอดบน CAM4 ด้วย SplitCam

CAM4 เป็นแพลตฟอร์ม cam-and-earn ระดับโลกพร้อมตัวควบคุมภูมิภาคในตัว — คุณสามารถซ่อนการถ่ายทอดในประเทศที่เลือก การถ่ายทอดผ่าน SplitCam ฟรีเป็น encoder ภายนอกเปิดการสลับฉากและโอเวอร์เลย์ที่เครื่องมือถ่ายทอดพื้นฐานไม่มี

คำตอบด่วน

คู่มือวิดีโอ

ทีละขั้นตอน

1
ดาวน์โหลดและติดตั้ง SplitCam

SplitCam คือซอฟต์แวร์สตรีมมิ่งสดฟรีสำหรับ Windows และ macOS ดาวน์โหลดและรันตัวติดตั้ง — ไม่ต้องสมัคร ไม่ต้องใช้บัตร ไม่มีลายน้ำ ไม่มีขีดจำกัดเวลา SplitCam คือ encoder ที่ส่งวิดีโอของคุณไปยัง CAM4

2
ตั้งค่ากล้องและฉาก

เปิด SplitCam และเพิ่มเว็บแคม สร้างฉากตามที่คุณต้องการให้ผู้ชมเห็น — โอเวอร์เลย์ ข้อความ กล้องที่สองหรือโทรศัพท์ ฟิลเตอร์ความงามหรือพื้นหลัง AI ทุกอย่างจะถูกนำมาใช้แบบสดก่อนที่สตรีมจะออกจากพีซีของคุณ

3
รับสตรีมคีย์ CAM4 ของคุณ

บน CAM4 คลิก Broadcast → Broadcast & Earn Money → Start Broadcast จากนั้น External Encoder ด้านบน กรอกวันเกิด เพศ และประเทศ ใช้ Get Stream Key และคัดลอก แถบเลื่อนสีเขียวใน Stream Settings ของ SplitCam ยืนยันการเชื่อมต่อ

4
เชื่อมต่อ SplitCam กับ CAM4

ใน SplitCam เปิด Stream Settings วาง URL เซิร์ฟเวอร์ CAM4 และสตรีมคีย์ในช่อง RTMP กำหนดเอง บิตเรต: 3,500–6,000 Kbps สำหรับ 1080p, 2,000–4,000 Kbps สำหรับ 720p เรียกใช้การทดสอบความเร็วในตัวก่อน

5
คลิก Go Live

กด Go Live ใน SplitCam จากนั้นเริ่มการถ่ายทอดสดบน CAM4 ภายในประมาณ 10 วินาที กล้องของคุณก็จะออนไลน์ การสตรีมครั้งต่อๆ ไปทำได้ด้วยคลิกเดียว — เปิด SplitCam, Go Live

เคล็ดลับ

ตั้งค่าข้อจำกัดทางภูมิภาค

CAM4 อนุญาตให้ซ่อนการถ่ายทอดในประเทศและภูมิภาคที่เฉพาะเจาะจง — ตั้งค่าบน CAM4 ก่อน go-live

ดูแถบเลื่อนสีเขียว

การตั้งค่า CAM4 แสดงแถบเลื่อนสีเขียวใน Stream Settings ของ SplitCam เมื่อคีย์ได้รับการยอมรับ — แดง = ตรวจคีย์

บิตเรตต่ำกว่าปกติ

CAM4 ingest จำกัดบิตเรตวิดีโอที่ประมาณ 3,000 Kbps — ต่ำกว่าไซต์ cam ส่วนใหญ่ อย่าดันสูงกว่า

ใช้การเชื่อมต่อแบบมีสาย

อีเทอร์เน็ตชนะ Wi-Fi ในการสตรีมที่ยาว — เฟรมที่หายคือทิปที่หาย ลากสายไปยังพีซีสตรีม

การแก้ไขปัญหา

สตรีม CAM4 ของคุณกระตุกหรือบัฟเฟอร์

เกือบทุกครั้งบิตเรตสูงกว่าที่อัปโหลดของคุณรับไหว รันการทดสอบความเร็วในตัวของ SplitCam แล้วตั้งบิตเรตที่ประมาณ 75% ของอัปโหลดที่วัดได้ — 3,500–6,000 Kbps สำหรับ 1080p ต่ำกว่าสำหรับ 720p อาการกระตุกจะหายไปเมื่อ encoder หยุดวิ่งแซงการเชื่อมต่อของคุณ

เฟรมหลุดระหว่างการถ่ายทอดบน CAM4

เฟรมหลุดหมายความว่าแพ็กเก็ตไปไม่ถึง CAM4 ทันเวลา — มักเป็น Wi-Fi ไม่เสถียร เปลี่ยนเป็นการเชื่อมต่อ Ethernet แบบมีสาย ปิดแอปที่กินแบนด์วิดท์ และลดบิตเรตลงเล็กน้อย พีคครั้งเดียวไม่เป็นไร แต่การเพิ่มขึ้นต่อเนื่องหมายความว่าการเชื่อมต่อตามไม่ทัน

จอดำ — ผู้ชมไม่เห็นวิดีโอบน CAM4

กล้องของคุณไม่ได้ถูกเลือกเป็นแหล่งที่ใช้งานใน SplitCam หรือมีแอปอื่นยึดไว้ ปิด Zoom, Skype หรือ OBS เลือกเว็บแคมใหม่ในรายการแหล่งของ SplitCam และยืนยันว่าตัวอย่างแสดงภาพของคุณก่อนกด Go Live

CAM4 ปฏิเสธสตรีมคีย์หรือไม่เชื่อมต่อ

คัดลอกสตรีมคีย์ใหม่ — ช่องว่างที่ท้ายหรือคีย์เก่าที่หมุนเปลี่ยนแล้วเป็นสาเหตุปกติ ยืนยันว่า URL เซิร์ฟเวอร์ตรงกับที่ CAM4 แสดง และเปิดใช้งานการถ่ายทอดผ่าน encoder ภายนอกในบัญชีของคุณแล้ว แถบเลื่อนสีเขียวใน Stream Settings ของ SplitCam ยืนยันว่าคีย์ถูกต้อง

ไม่มีเสียงหรือเสียงไม่ตรงกับภาพบน CAM4

เลือก SplitCam เป็นทั้งกล้องและไมโครโฟน และเลือกไมโครโฟนจริงของคุณในแหล่งเสียงของ SplitCam หากเสียงตามหลังวิดีโอ ลดความละเอียดลงหนึ่งระดับ — encoder ทำงานหนักเกินไปและเสียงกำลังรอเฟรมที่มาช้า

คำถามที่พบบ่อย

CAM4 รองรับ OBS / encoder ภายนอกอย่างเป็นทางการหรือไม่?

ใช่ — CAM4 มี OBS Guide อย่างเป็นทางการบนไซต์สนับสนุนและแนะนำ External Encoder เพื่อประสบการณ์ที่ดีที่สุด SplitCam ใช้เส้นทาง RTMP เดียวกัน

ฉันสามารถบล็อกสตรีม CAM4 ตามภูมิภาคได้หรือไม่?

ใช่ — CAM4 มีการจำกัดตามภูมิภาคในตัวเพื่อซ่อนการถ่ายทอดในบางประเทศ ตั้งค่าใน CAM4 ไม่ใช่ใน SplitCam

คีย์สตรีม CAM4 อยู่ที่ไหน?

Broadcast → Broadcast & Earn Money → Start Broadcast → External Encoder → Get Stream Key

บิตเรตสำหรับ CAM4?

ต่ำกว่าค่าเฉลี่ย — CAM4 ingest จำกัดที่ ~3,000 Kbps ที่ 30 fps พร้อม keyframe 1 วินาที ตารางทางการแนะนำไม่เกิน ~3,000
