Streamate

Як транслювати на Streamate зі SplitCam

Streamate — усталена кам-платформа, і вона є наперед налаштованою ціллю в SplitCam, у списку каналів, тож налаштування швидше за ручний ввід RTMP: виберіть Streamate, вставте ключ, готово.

Швидка відповідь
Трансляція на Streamate зі SplitCam: встановіть SplitCam, зберіть сцену, на Streamate скористайтеся SM Connect → Start Show і скопіюйте ключ, далі у SplitCam відкрийте Stream Settings → Add Channel → Streamate і вставте.

Відеопосібник

Покроково

1
Завантажте та встановіть SplitCam

SplitCam — безкоштовна програма для трансляцій на Windows і macOS. Завантажте й запустіть інсталятор — без реєстрації, без картки, без водяного знака, без обмеження часу. Це кодувальник, що надсилає ваше відео на Streamate.

2
Налаштуйте камеру й сцену

Відкрийте SplitCam і додайте вебкамеру. Зберіть сцену так, як хочете, щоб її бачили глядачі — оверлеї, текст, друга камера чи телефон, beauty-фільтри або AI-фон. Усе застосовується наживо, перш ніж стрим покине ваш ПК.

3
Отримайте ключ трансляції Streamate

На Streamate відкрийте SM Connect, погодьтеся з умовами, натисніть Start Show ліворуч і закрийте вікно, що відкрилося — далі скопіюйте ключ стримінгу. У SplitCam відкрийте Stream Settings → Add Channel, виберіть Streamate зі списку й вставте ключ. Зелений повзунок означає з'єднання.

4
Під'єднайте SplitCam до Streamate

У SplitCam відкрийте Stream Settings, вставте URL сервера й ключ трансляції Streamate у поля кастомного RTMP. Задайте бітрейт 3500–6000 Кбіт/с для 1080p, 2000–4000 Кбіт/с для 720p і спершу запустіть вбудований тест швидкості.

5
Натисніть Go Live

Натисніть Go Live у SplitCam, далі почніть трансляцію на Streamate. За ~10 секунд ваша камера наживо. Наступні трансляції — один клік: відкрийте SplitCam, Go Live.

Поради для профі

Streamate інтегрований

Без ручного URL RTMP — у SplitCam Streamate є у списку Add Channel; просто виберіть і вставте ключ.

Закрийте попап SM Connect

Після Start Show Streamate відкриває вікно — закрийте; воно лише для показу ключа стримінгу.

Зафіксуйте роздільну здатність на 1080p

Поле Video Resolution у SM Connect може нишком стрибнути на 1440, що насправді не надсилається й тихо знижує вашу якість — задайте вручну 1080p. Падіння бітрейту при нерухомому зображенні нормальне: стрим Streamate адаптивний.

Використовуйте дротове з'єднання

Ethernet перемагає Wi-Fi під час довгих трансляцій — втрачені кадри означають втрачені чайові. Підключіть кабель до комп'ютера, що веде ефір.

Спершу зробіть приватний тест

Проведіть коротку тестову трансляцію, щоб перевірити камеру, звук, кадрування та оверлеї, перш ніж відкривати кімнату для глядачів.

Усунення несправностей

Трансляція Streamate гальмує або буферизується

Майже завжди бітрейт заданий вище, ніж витримає ваш аплоуд. Запустіть вбудований тест швидкості SplitCam, далі задайте бітрейт приблизно 75% від виміряного аплоуду — 3500–6000 Кбіт/с для 1080p, нижче для 720p. Гальмування зникає, коли кодувальник перестає перевищувати ваш канал.

Втрата кадрів під час трансляції Streamate

Втрата кадрів означає, що пакети не доходять до Streamate вчасно — зазвичай нестабільний Wi-Fi. Перейдіть на дротовий Ethernet, закрийте застосунки, що з'їдають канал, і трохи знизьте бітрейт. Один стрибок — це нічого; постійне зростання означає, що канал не встигає.

Чорний екран — глядачі не бачать відео на Streamate

Ваша камера не вибрана як активне джерело у SplitCam, або її тримає інший застосунок. Закрийте Zoom, Skype чи OBS, виберіть вебкамеру знову у списку джерел SplitCam і переконайтеся, що попередній перегляд показує ваше зображення, перш ніж натиснути Go Live.

Streamate відхиляє ключ трансляції або не під'єднується

Скопіюйте ключ трансляції знову — пробіл у кінці чи старий, скинутий ключ — найчастіші причини. Переконайтеся, що URL сервера збігається з тим, що показує Streamate, і що трансляція зовнішнім кодувальником увімкнена на акаунті. Зелений повзунок у Stream Settings SplitCam підтверджує дійсний ключ.

Немає звуку або звук розсинхронізований на Streamate

Виберіть SplitCam камерою І мікрофоном, а в джерелі звуку SplitCam виберіть свій справжній мікрофон. Якщо звук відстає від відео, знизьте роздільну здатність на один рівень — кодувальник перевантажений, і звук чекає на запізнілі кадри.

Часті питання

Чи інтегрований Streamate у SplitCam?

Так — Streamate з'являється у списку Add Channel SplitCam; ви його вибираєте, а не вводите URL RTMP вручну.

Де ключ стримінгу Streamate?

SM Connect → погодьтеся з умовами → Start Show → закрийте попап → скопіюйте ключ.

Який бітрейт для Streamate?

Streamate не задає жорсткої стелі — 3500–6000 Кбіт/с у 1080p працює добре. Стрим адаптивний, тож нижчий бітрейт при нерухомому зображенні нормальний, а не баг.

Чи безкоштовний SplitCam для Streamate?

Так — безкоштовний, без водяного знака й без обмеження часу; а оскільки Streamate — інтегрований канал у SplitCam, окремих витрат на кодувальник немає.
