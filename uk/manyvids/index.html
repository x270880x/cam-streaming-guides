Як транслювати на MV Live зі SplitCam

ManyVids — платформа креатор-економіки: продаж кліпів, custom-відео, fan-club підписки та продукт прямих трансляцій MV Live. Стандартний браузерний нададач Creator Studio працює у вебі, але також надає стандартний шлях External Encoder, до якого підключається безкоштовний SplitCam — він дозволяє транслювати багатокамерні сцени, оверлеї та фільтри на тій самій платформі, дружній до креаторів.

Швидка відповідь
Трансляція на MV Live зі SplitCam: встановіть SplitCam, зберіть сцену, у Creator Studio відкрийте MV Live → Broadcast Settings → External Encoder, скопіюйте URL сервера й stream key, вставте у SplitCam, Go Live.

Покроково

1
Завантажте та встановіть SplitCam

SplitCam — безкоштовне ПЗ для прямих трансляцій для Windows і macOS — без реєстрації, без картки, без вотермарки.

2
Зберіть сцену

Відкрийте SplitCam і додайте камеру. Додайте оверлеї, тексти, другу камеру або телефон, beauty-фільтри чи AI-фон — ідеально для MV Live goal-reveal збірок і тригерів винагород.

3
Отримайте URL і stream key MV Live

Увійдіть в акаунт креатора ManyVids, відкрийте Creator Studio, перейдіть до MV Live → Broadcast Settings → External Encoder. Сторінка покаже URL сервера та унікальний stream key. Скопіюйте обидва.

4
Підключіть SplitCam до MV Live

У SplitCam відкрийте Stream Settings, вставте URL і stream key MV Live у кастомні поля RTMP. Поставте бітрейт 3500–6000 Кбіт/с при 1920×1080, 30 fps, ключовий кадр кожні 2 секунди.

5
Натисніть Go Live

Натисніть Go Live у SplitCam, потім запустіть трансляцію MV Live з Creator Studio. Приблизно за 10 секунд ваш стрім доходить до аудиторії MV Live.

Поради для профі

Креатор-економіка, не лише live

ManyVids — не чиста cam-платформа: MV Live — це один із потоків доходу поряд із продажем кліпів, custom-відео та fan-club підписками. Використовуйте прямі трансляції, щоб направляти глядачів до інших способів монетизації.

Токенові чайові всередині MV Live

MV Live має власну систему токенових чайових усередині live-кімнати. Плануйте меню цілей і тригери винагород подібно до Chaturbate / Stripchat — на існуючій аудиторії ManyVids це конвертує добре.

Браузер vs зовнішній енкодер

Вбудований браузерний нададач Creator Studio — однокамерний. SplitCam через External Encoder відкриває багатокамерні сцени, оверлеї та фільтри.

Використовуйте дротове з'єднання

Ethernet перемагає Wi-Fi під час довгих трансляцій — втрачені кадри означають втрачені чайові. Підключіть кабель до комп'ютера, що веде ефір.

Усунення несправностей

Трансляція ManyVids гальмує або буферизується

Майже завжди бітрейт заданий вище, ніж витримає ваш аплоуд. Запустіть вбудований тест швидкості SplitCam, далі задайте бітрейт приблизно 75% від виміряного аплоуду — 3500–6000 Кбіт/с для 1080p, нижче для 720p. Гальмування зникає, коли кодувальник перестає перевищувати ваш канал.

Втрата кадрів під час трансляції ManyVids

Втрата кадрів означає, що пакети не доходять до ManyVids вчасно — зазвичай нестабільний Wi-Fi. Перейдіть на дротовий Ethernet, закрийте застосунки, що з'їдають канал, і трохи знизьте бітрейт. Один стрибок — це нічого; постійне зростання означає, що канал не встигає.

Чорний екран — глядачі не бачать відео на ManyVids

Ваша камера не вибрана як активне джерело у SplitCam, або її тримає інший застосунок. Закрийте Zoom, Skype чи OBS, виберіть вебкамеру знову у списку джерел SplitCam і переконайтеся, що попередній перегляд показує ваше зображення, перш ніж натиснути Go Live.

ManyVids відхиляє ключ трансляції або не під'єднується

Скопіюйте ключ трансляції знову — пробіл у кінці чи старий, скинутий ключ — найчастіші причини. Переконайтеся, що URL сервера збігається з тим, що показує ManyVids, і що трансляція зовнішнім кодувальником увімкнена на акаунті. Зелений повзунок у Stream Settings SplitCam підтверджує дійсний ключ.

Немає звуку або звук розсинхронізований на ManyVids

Виберіть SplitCam камерою І мікрофоном, а в джерелі звуку SplitCam виберіть свій справжній мікрофон. Якщо звук відстає від відео, знизьте роздільну здатність на один рівень — кодувальник перевантажений, і звук чекає на запізнілі кадри.

Часті питання

Чи підтримує MV Live (ManyVids) зовнішні енкодери, як SplitCam?

Так — секція MV Live у Creator Studio має опцію External Encoder у Broadcast Settings. Стандартний URL сервера RTMP і stream key; OBS, SplitCam та vMix підключаються без проблем.

Де отримати stream key MV Live?

Creator Studio → MV Live → Broadcast Settings → External Encoder. Як URL сервера, так і stream key з'являються там разом — скопіюйте обидва в кастомні поля RTMP у SplitCam.

Який бітрейт для MV Live?

Виштовхуйте 1920×1080 при 30 fps, 3500–6000 Кбіт/с із ключовим кадром кожні 2 секунди. Спершу запустіть вбудований тест швидкості SplitCam.

Чи безкоштовний SplitCam для MV Live?

Так — SplitCam безкоштовний, без вотермарки та без обмеження часу. Опція External Encoder у Creator Studio ManyVids безкоштовна.
