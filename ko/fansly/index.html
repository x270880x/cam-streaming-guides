SplitCam으로 Fansly Live 방송하기

Fansly는 OnlyFans의 직접 경쟁자 — 콘텐츠 룰이 더 느슨하고 크리에이터 베이스가 빠르게 성장 중이며, 구독, PPV 콘텐츠 그리고 라이브 방송 제품 Fansly Live를 제공합니다. 기본은 브라우저 방송기지만 Creator Dashboard에서 표준 External Encoder 경로도 열어줘 무료 SplitCam이 그대로 접속됩니다 — 구독자 베이스에 멀티 카메라, 오버레이, 필터 장착 방송을 송출할 수 있습니다.

빠른 답변
SplitCam으로 Fansly Live 방송: SplitCam 설치, 장면 구성, Creator Dashboard에서 Live → Broadcast Settings → External Encoder 열기, 서버 URL과 stream key 복사 후 SplitCam에 붙여넣기, Go Live.

단계별 안내

1
SplitCam 다운로드 및 설치

SplitCam은 Windows와 macOS용 무료 방송 소프트웨어 — 가입 없음, 카드 없음, 워터마크 없음.

2
장면 구성

SplitCam에서 웹캠을 추가하고 오버레이, 텍스트, 두 번째 카메라나 폰, 뷰티 필터, AI 배경을 얹으세요 — 유료 구독자의 기대치에 맞는 정돈된 장면을 만들 수 있습니다.

3
Fansly URL과 stream key 받기

Fansly 크리에이터 계정으로 로그인해 Creator Dashboard를 열고 Live → Broadcast Settings → External Encoder로 이동. 서버 URL과 고유 stream key가 표시됩니다. 두 값을 복사.

4
SplitCam을 Fansly에 연결

SplitCam의 Stream Settings를 열고 Fansly 서버 URL과 stream key를 커스텀 RTMP 필드에 붙여넣으세요. 비트레이트는 3,500–6,000 Kbps, 1920×1080, 30 fps, 키프레임 2초.

5
Go Live 클릭

SplitCam에서 Go Live를 누른 뒤 Creator Dashboard에서 Fansly Live 방송을 시작. 약 10초 안에 신호가 Fansly 구독자에게 도달합니다.

프로 팁

구독자 우선 오디언스

Fansly의 오디언스는 구독 기반 — 라이브를 보는 사람은 이미 매달 결제하는 사람들입니다. 공개방 지표를 좇기보다 로열티를 보상하는 콘텐츠(독점 Q&A, 비하인드, 커스텀 팁 골)를 설계하세요.

구독에 더해지는 팁

Fansly Live는 기본 구독 위에 스트림 내 팁도 지원. 자리잡은 크리에이터에게는 구독 + 팁 합산 매출이 순 cam 플랫폼의 팁 단독보다 큰 경우가 많습니다.

브라우저 방송 vs 외부

기본 브라우저 방송은 싱글 소스. SplitCam을 External Encoder로 연결하면 멀티 카메라, 오버레이, 뷰티 필터, AI 배경이 풀려 유료 구독 콘텐츠 퀄리티에 맞출 수 있습니다.

유선 연결을 쓰세요

긴 방송에서는 이더넷이 와이파이를 이깁니다 — 프레임이 끊기면 팁도 끊깁니다. 방송 PC에 랜선을 연결하세요.

문제 해결

Fansly 방송이 끊기거나 버퍼링됨

거의 항상 비트레이트가 업로드가 감당할 수 있는 것보다 높게 설정된 경우입니다. SplitCam 내장 속도 테스트를 돌린 뒤 비트레이트를 측정된 업로드의 약 75%로 설정하세요 — 1080p에 3,500–6,000 Kbps, 720p는 더 낮게. 인코더가 회선을 초과하지 않으면 끊김이 사라집니다.

Fansly 방송 중 프레임 끊김

프레임 끊김은 패킷이 Fansly에 제때 도착하지 못한다는 뜻입니다 — 보통 불안정한 와이파이. 유선 이더넷으로 바꾸고 대역폭을 먹는 앱을 닫고 비트레이트를 약간 낮추세요. 한 번 튀는 건 괜찮지만 계속 늘면 회선이 못 따라가는 것입니다.

검은 화면 — 시청자가 Fansly에서 영상을 못 봄

카메라가 SplitCam에서 활성 소스로 선택되지 않았거나 다른 앱이 잡고 있습니다. Zoom, Skype, OBS를 닫고 SplitCam 소스 목록에서 웹캠을 다시 고르고 Go Live 전에 미리보기에 영상이 나오는지 확인하세요.

Fansly이 스트림 키를 거부하거나 연결되지 않음

스트림 키를 다시 복사하세요 — 끝의 공백이나 재설정된 오래된 키가 가장 흔한 원인입니다. 서버 URL이 Fansly이 표시한 것과 일치하고 계정에 외부 인코더 방송이 켜져 있는지 확인하세요. SplitCam Stream Settings의 초록 슬라이더가 유효한 키를 확인해 줍니다.

Fansly에서 소리가 없거나 음성이 어긋남

SplitCam을 카메라 그리고 마이크로 모두 선택하고 SplitCam 오디오 소스 안에서 진짜 마이크를 고르세요. 오디오가 영상보다 뒤처지면 해상도를 한 단계 낮추세요 — 인코더가 과부하라 오디오가 늦은 프레임을 기다리는 것입니다.

자주 묻는 질문

Fansly Live는 SplitCam 같은 외부 인코더를 지원하나요?

지원합니다 — Creator Dashboard의 Live 섹션 Broadcast Settings 아래에 External Encoder 옵션이 있습니다. 표준 RTMP 서버 URL과 stream key, OBS, SplitCam, vMix 모두 연결됩니다.

Fansly stream key는 어디 있나요?

Creator Dashboard → Live → Broadcast Settings → External Encoder. 서버 URL과 stream key 둘 다 여기에 표시됩니다. 둘 다 SplitCam의 커스텀 RTMP 필드에 복사.

Fansly Live 비트레이트는 얼마?

1920×1080, 30 fps, 3,500–6,000 Kbps, 키프레임 2초. SplitCam 내장 속도 테스트를 먼저 돌리세요.

SplitCam은 Fansly에서 무료인가요?

네 — SplitCam은 무료, 워터마크 없음, 시간 제한 없음. Fansly의 Creator Dashboard에서 열어주는 외부 인코더 옵션도 무료입니다.

Fansly에서 모델은 얼마나 벌 수 있나요?

Fansly에서의 수입은 시청자 규모, 방송 시간, 팁 문화에 따라 달라집니다. 활발한 방송인은 보통 월 $200~$3,000을 가져가며, 상위권은 $10,000 이상을 벌기도 합니다. 수익 배분은 Fansly의 수수료 구조를 따르므로, 라이브 시작 전에 모델 계약서를 반드시 확인하세요.

Fansly은(는) 방송인에게 안전한가요?

Fansly은(는) 지급 전에 연령 및 신분 확인을 요구하므로 모델을 사기로부터 보호합니다. 활동명을 사용하고, 카메라 앞에서 개인 정보를 절대 공유하지 말며, 지역 차단을 활성화해 본인이 거주하는 지역에서 방송이 보이지 않도록 하고, 모든 시청자 요청을 거래로 다루세요. SplitCam의 오버레이와 AI 배경 기능을 사용하면 실제 주변 환경을 가리거나 교체할 수도 있습니다.

Fansly에서 모델이 되려면 어떤 서류가 필요한가요?

Fansly은(는) 일반적으로 정부 발급 사진 신분증(여권, 운전면허증 또는 신분증), 신분증을 들고 찍은 셀카, 그리고 세금/지급 양식(미국은 W-9, 미국 외 지역은 W-8BEN)을 요구합니다. 승인까지 보통 24~72시간이 걸리며, 승인되면 당일에 바로 라이브를 시작할 수 있습니다.

휴대폰으로 Fansly에 방송할 수 있나요?

Fansly은(는) 보통 모바일 방송 앱이나 모바일 웹 방송 기능을 제공하지만, 오버레이, 보조 카메라, AI 배경이 없는 등 기능이 제한적입니다. 완전한 제작 품질을 원한다면 컴퓨터에서 SplitCam으로 방송하고, 휴대폰을 보조 카메라로 사용하세요(SplitCam은 휴대폰에서 IP 카메라 입력을 받을 수 있습니다).

Fansly은(는) OBS나 외부 인코더를 지원하나요?

네 — Fansly은(는) 방송인 패널에서 RTMP 서버 URL과 스트림 키를 제공합니다. 둘 다 SplitCam의 Stream Settings → Custom RTMP에 붙여 넣고, 1920×1080, 30fps, 4,000~5,000 Kbps 비트레이트로 설정한 뒤 Go Live를 클릭하세요. Custom RTMP 경로를 사용하면 SplitCam의 모든 장면 구성 기능(멀티 카메라, 오버레이, 필터)을 그대로 활용할 수 있습니다.
