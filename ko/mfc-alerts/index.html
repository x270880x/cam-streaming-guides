방송에 MFC Alerts 추가하는 방법

영상 가이드

단계별 안내

1
SplitCam 설치 및 웹캠 추가

Windows나 macOS용 무료 SplitCam을 설치한 뒤 웹캠을 소스로 추가하세요. MFC Alerts는 이 카메라 위 레이어로 놓입니다.

2
Browser 탭 열고 mfcalerts.com 접속

SplitCam에서 Browser 탭을 열고 www.mfcalerts.com에 접속하세요. 로그인하거나, mfcalerts.com 계정이 없으면 가입하세요.

3
알림 URL 복사

mfcalerts.com에서 Copy to clipboard로 개인 알림 URL을 복사하세요 — 팁 애니메이션을 렌더링하는 페이지입니다.

4
URL을 Browser 레이어로 추가 — 웹캠 위

URL을 SplitCam Browser 창에 붙여넣고 Add를 누르세요. 그다음 소스 목록을 정리해 MFC Alerts가 웹캠 위에 오게 하세요(점 3개 메뉴 → Move Up). 아래에 있으면 효과가 숨겨집니다.

5
테스트 팁으로 점검

Settings → Send test tip을 열어 알림 효과가 카메라 위에 나타나는지 확인하세요. 그다음 MyFreeCams에서 평소처럼 방송하면 — 실제 팁이 애니메이션을 일으킵니다.

프로 팁

MFC Alerts는 웹캠 위에 있어야 함

가장 흔한 실수: 소스 목록에서 MFC Alerts 레이어가 웹캠 아래에 있으면 효과가 숨겨집니다. 위로 옮기세요.

mfcalerts.com 계정 필요

알림 URL은 개인용입니다 — 계정이 없으면 먼저 mfcalerts.com에 가입하세요.

라이브 전에 테스트 팁을 보내세요

Settings → Send test tip으로 오버레이가 작동하는지 확인하세요 — 쇼 도중에 오류를 발견하지 마세요.

유선 연결을 쓰세요

긴 방송에서는 이더넷이 와이파이를 이깁니다 — 프레임이 끊기면 팁도 끊깁니다. 방송 PC에 랜선을 연결하세요.

문제 해결

MFC Alerts 방송이 끊기거나 버퍼링됨

거의 항상 비트레이트가 업로드가 감당할 수 있는 것보다 높게 설정된 경우입니다. SplitCam 내장 속도 테스트를 돌린 뒤 비트레이트를 측정된 업로드의 약 75%로 설정하세요 — 1080p에 3,500–6,000 Kbps, 720p는 더 낮게. 인코더가 회선을 초과하지 않으면 끊김이 사라집니다.

MFC Alerts 방송 중 프레임 끊김

프레임 끊김은 패킷이 MFC Alerts에 제때 도착하지 못한다는 뜻입니다 — 보통 불안정한 와이파이. 유선 이더넷으로 바꾸고 대역폭을 먹는 앱을 닫고 비트레이트를 약간 낮추세요. 한 번 튀는 건 괜찮지만 계속 늘면 회선이 못 따라가는 것입니다.

검은 화면 — 시청자가 MFC Alerts에서 영상을 못 봄

카메라가 SplitCam에서 활성 소스로 선택되지 않았거나 다른 앱이 잡고 있습니다. Zoom, Skype, OBS를 닫고 SplitCam 소스 목록에서 웹캠을 다시 고르고 Go Live 전에 미리보기에 영상이 나오는지 확인하세요.

MFC Alerts이 스트림 키를 거부하거나 연결되지 않음

스트림 키를 다시 복사하세요 — 끝의 공백이나 재설정된 오래된 키가 가장 흔한 원인입니다. 서버 URL이 MFC Alerts이 표시한 것과 일치하고 계정에 외부 인코더 방송이 켜져 있는지 확인하세요. SplitCam Stream Settings의 초록 슬라이더가 유효한 키를 확인해 줍니다.

MFC Alerts에서 소리가 없거나 음성이 어긋남

SplitCam을 카메라 그리고 마이크로 모두 선택하고 SplitCam 오디오 소스 안에서 진짜 마이크를 고르세요. 오디오가 영상보다 뒤처지면 해상도를 한 단계 낮추세요 — 인코더가 과부하라 오디오가 늦은 프레임을 기다리는 것입니다.

자주 묻는 질문

MFC Alerts가 무엇인가요?

MyFreeCams용 알림 시스템으로 시청자가 팁을 보낼 때 방송에 영상 효과를 표시합니다 — SplitCam에 Browser 오버레이로 추가됩니다.

내 MFC Alerts가 왜 안 보이나요?

거의 항상 레이어 순서입니다 — MFC Alerts Browser 레이어가 SplitCam 소스 목록에서 웹캠 위에 있어야 합니다.

MFC Alerts에 계정이 필요한가요?

네 — 개인 알림 URL을 받으려면 mfcalerts.com에 가입하세요.

이걸 위해 SplitCam은 무료인가요?

네 — SplitCam은 무료, 워터마크 없음, 시간 제한 없음이고 MFC Alerts 브라우저 오버레이가 그 안에서 추가 비용 없이 작동합니다.
