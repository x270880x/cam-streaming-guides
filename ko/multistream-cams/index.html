영상 가이드

단계별 안내

1
SplitCam 설치

SplitCam은 멀티스트림이 내장된 Windows와 macOS용 무료 라이브 방송 소프트웨어입니다. 설치하세요 — 워터마크 없음, 가입 없음.

2
카메라와 장면 설정

SplitCam을 열고 웹캠을 추가한 뒤 오버레이와 필터로 장면을 구성하세요. 한 장면이 모든 대상을 공급합니다.

3
각 캠 사이트에서 RTMP 키 받기

각 캠 플랫폼에서 외부 / RTMP 방송을 켜고 서버 URL과 스트림 키를 복사하세요. 방송할 각 사이트마다 반복하세요 — 정확한 경로는 각 플랫폼 가이드를 보세요.

4
SplitCam에 각 대상 추가

Stream Settings를 열고 각 캠 사이트를 커스텀 RTMP 대상으로 추가하세요 — 서버 URL과 키를 붙여넣습니다. 라이브할 곳을 모두 체크하세요.

5
Go Live 한 번 클릭

Go Live를 누르세요. SplitCam이 하나의 인코딩에서 선택한 각 캠 사이트로 동시에, peer-to-peer로 스트림을 보냅니다 — 추가 비용 없이.

프로 팁

당신의 업로드를 주시하세요

멀티스트림은 업로드 부하를 배가합니다. 각 대상이 자체 비트레이트를 씁니다 — 회선이 합계를 감당하는지 확인하세요.

플랫폼 규칙을 확인하세요

일부 캠 사이트는 다른 곳에서 동시 방송을 금지합니다 — 멀티스트림 전에 확인하세요.

랜선 — 여기선 끊기면 안 됨

멀티스트림은 업로드 부하를 배가하므로 와이파이가 한 번 끊기면 모든 대상이 동시에 떨어질 수 있습니다. 여기서 랜선은 선택이 아닙니다.

상태 모니터를 보세요

SplitCam은 대상별 상태를 표시합니다 — 업로드가 못 버티면 한 사이트를 떼세요.

문제 해결

Multiple Cam Sites 방송이 끊기거나 버퍼링됨

거의 항상 비트레이트가 업로드가 감당할 수 있는 것보다 높게 설정된 경우입니다. SplitCam 내장 속도 테스트를 돌린 뒤 비트레이트를 측정된 업로드의 약 75%로 설정하세요 — 1080p에 3,500–6,000 Kbps, 720p는 더 낮게. 인코더가 회선을 초과하지 않으면 끊김이 사라집니다.

Multiple Cam Sites 방송 중 프레임 끊김

프레임 끊김은 패킷이 Multiple Cam Sites에 제때 도착하지 못한다는 뜻입니다 — 보통 불안정한 와이파이. 유선 이더넷으로 바꾸고 대역폭을 먹는 앱을 닫고 비트레이트를 약간 낮추세요. 한 번 튀는 건 괜찮지만 계속 늘면 회선이 못 따라가는 것입니다.

검은 화면 — 시청자가 Multiple Cam Sites에서 영상을 못 봄

카메라가 SplitCam에서 활성 소스로 선택되지 않았거나 다른 앱이 잡고 있습니다. Zoom, Skype, OBS를 닫고 SplitCam 소스 목록에서 웹캠을 다시 고르고 Go Live 전에 미리보기에 영상이 나오는지 확인하세요.

Multiple Cam Sites이 스트림 키를 거부하거나 연결되지 않음

스트림 키를 다시 복사하세요 — 끝의 공백이나 재설정된 오래된 키가 가장 흔한 원인입니다. 서버 URL이 Multiple Cam Sites이 표시한 것과 일치하고 계정에 외부 인코더 방송이 켜져 있는지 확인하세요. SplitCam Stream Settings의 초록 슬라이더가 유효한 키를 확인해 줍니다.

Multiple Cam Sites에서 소리가 없거나 음성이 어긋남

SplitCam을 카메라 그리고 마이크로 모두 선택하고 SplitCam 오디오 소스 안에서 진짜 마이크를 고르세요. 오디오가 영상보다 뒤처지면 해상도를 한 단계 낮추세요 — 인코더가 과부하라 오디오가 늦은 프레임을 기다리는 것입니다.

자주 묻는 질문

SplitCam 멀티스트림은 무료인가요?

네 — 멀티스트림은 내장이며 무료, 대상별 요금 없음, 워터마크 없음.

동시에 몇 개의 캠 사이트에 방송할 수 있나요?

업로드 대역폭이 감당하는 만큼 — 각 대상이 자체 비트레이트를 씁니다.

클라우드 릴레이를 쓰나요?

아니오 — SplitCam은 PC에서 각 플랫폼 ingest로 직접 peer-to-peer로 스트림을 보냅니다.

멀티스트림이 PC를 느리게 하나요?

인코딩은 한 번 하고 재사용합니다; 하드웨어 인코딩이 CPU 부하를 낮게 유지합니다. 업로드 대역폭이 진짜 한계입니다.

Multiple Cam Sites에서 모델은 얼마나 벌 수 있나요?

Multiple Cam Sites에서의 수입은 시청자 규모, 방송 시간, 팁 문화에 따라 달라집니다. 활발한 방송인은 보통 월 $200~$3,000을 가져가며, 상위권은 $10,000 이상을 벌기도 합니다. 수익 배분은 Multiple Cam Sites의 수수료 구조를 따르므로, 라이브 시작 전에 모델 계약서를 반드시 확인하세요.

Multiple Cam Sites은(는) 방송인에게 안전한가요?

Multiple Cam Sites은(는) 지급 전에 연령 및 신분 확인을 요구하므로 모델을 사기로부터 보호합니다. 활동명을 사용하고, 카메라 앞에서 개인 정보를 절대 공유하지 말며, 지역 차단을 활성화해 본인이 거주하는 지역에서 방송이 보이지 않도록 하고, 모든 시청자 요청을 거래로 다루세요. SplitCam의 오버레이와 AI 배경 기능을 사용하면 실제 주변 환경을 가리거나 교체할 수도 있습니다.

Multiple Cam Sites에서 모델이 되려면 어떤 서류가 필요한가요?

Multiple Cam Sites은(는) 일반적으로 정부 발급 사진 신분증(여권, 운전면허증 또는 신분증), 신분증을 들고 찍은 셀카, 그리고 세금/지급 양식(미국은 W-9, 미국 외 지역은 W-8BEN)을 요구합니다. 승인까지 보통 24~72시간이 걸리며, 승인되면 당일에 바로 라이브를 시작할 수 있습니다.

휴대폰으로 Multiple Cam Sites에 방송할 수 있나요?

Multiple Cam Sites은(는) 보통 모바일 방송 앱이나 모바일 웹 방송 기능을 제공하지만, 오버레이, 보조 카메라, AI 배경이 없는 등 기능이 제한적입니다. 완전한 제작 품질을 원한다면 컴퓨터에서 SplitCam으로 방송하고, 휴대폰을 보조 카메라로 사용하세요(SplitCam은 휴대폰에서 IP 카메라 입력을 받을 수 있습니다).

Multiple Cam Sites은(는) OBS나 외부 인코더를 지원하나요?

네 — Multiple Cam Sites은(는) 방송인 패널에서 RTMP 서버 URL과 스트림 키를 제공합니다. 둘 다 SplitCam의 Stream Settings → Custom RTMP에 붙여 넣고, 1920×1080, 30fps, 4,000~5,000 Kbps 비트레이트로 설정한 뒤 Go Live를 클릭하세요. Custom RTMP 경로를 사용하면 SplitCam의 모든 장면 구성 기능(멀티 카메라, 오버레이, 필터)을 그대로 활용할 수 있습니다.
