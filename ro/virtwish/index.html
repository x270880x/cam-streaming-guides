Cum transmiți pe VirtWish cu SplitCam

VirtWish e o platformă cam interactivă. Setările de transmisie îți dau un URL stream și o stream key separată într-o secțiune OBS — SplitCam gratuit le ia pe ambele și rulează show-ul cu scene și overlay-uri.

Pas cu pas

1
Descarcă și instalează SplitCam

SplitCam este software de streaming gratuit pentru Windows și macOS. Descarcă-l și rulează installerul — fără înregistrare, fără card, fără filigran, fără limită de timp. SplitCam este encoderul care îți trimite videoul către VirtWish.

2
Configurează camera și scena

Deschide SplitCam și adaugă webcamul. Construiește scena așa cum vrei să o vadă spectatorii — suprapuneri, text, o a doua cameră sau telefonul, filtre beauty sau fundal AI. Totul se aplică live înainte ca streamul să plece din PC.

3
Obține cheia de stream VirtWish

Pe VirtWish clic pe iconița din colțul dreapta sus → Profile → Start Broadcast, apoi Start Broadcast din nou pentru a ajunge la secțiunea OBS. Copiază linkul de pe primul rând în câmpul Stream URL al SplitCam, iar Stream Key separat în câmpul cheii.

4
Conectează SplitCam la VirtWish

În SplitCam deschide Stream Settings, lipește URL-ul serverului și cheia de stream VirtWish în câmpurile RTMP personalizate. Bitrate: 3.500–6.000 Kbps la 1080p, 2.000–4.000 Kbps la 720p. Rulează mai întâi testul de viteză integrat.

5
Apasă Go Live

Apasă Go Live în SplitCam, apoi pornește transmisiunea pe VirtWish. În ~10 secunde camera ta este live. Următoarele streamuri se pornesc cu un singur clic — deschizi SplitCam, Go Live.

Sfaturi

Link pe primul rând

Secțiunea OBS VirtWish pune URL-ul streamului pe primul rând — copiezi în Stream URL SplitCam, apoi cheia separată.

Două click-uri pe Start Broadcast

Apeși Start Broadcast de două ori pe VirtWish pentru a ajunge la secțiunea OBS — e așteptat, nu bug.

Folosește conexiune prin cablu

Ethernet bate Wi-Fi într-un stream lung — un frame pierdut e un tip pierdut. Trage un cablu până la PC-ul de stream.

Fă mai întâi un test privat

Rulează o transmisiune scurtă de test pentru cameră, audio, încadrare și overlay-uri înainte să deschizi camera publicului.

Întrebări frecvente

Unde sunt datele RTMP VirtWish?

Iconița dreapta sus → Profile → Start Broadcast de două ori → secțiunea OBS afișează linkul și stream key.

Suportă VirtWish encodere externe?

Da — secțiunea OBS oferă URL stream și cheie, deci SplitCam funcționează.

Ce bitrate pentru VirtWish?

VirtWish nu publică plafon oficial; 3.500–6.000 Kbps la 1080p e sigur. Potrivește rezoluția SplitCam cu cea setată pe VirtWish pentru a evita rescaling.

SplitCam e gratuit pentru VirtWish?

Da — gratuit, fără filigran și fără limită de timp. Setup-ul URL-și-cheie VirtWish costă doar minutele necesare.
