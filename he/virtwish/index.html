איך לשדר ב-VirtWish עם SplitCam

‏VirtWish היא פלטפורמת קאם אינטראקטיבית. הגדרות השידור נותנות לכם URL זרם ומפתח שידור נפרדים במקטע ה-OBS — SplitCam החינמי לוקח את שניהם ומריץ את המופע עם סצנות ושכבות.

תשובה מהירה
שידור ב-VirtWish עם SplitCam: התקינו SplitCam, בנו סצנה, ב-VirtWish פתחו Profile → Start Broadcast עד מקטע ה-OBS, העתיקו קישור ומפתח, הדביקו את שניהם ב-SplitCam, Go Live.

מדריך וידאו

שלב אחר שלב

1
הורידו והתקינו SplitCam

‏SplitCam היא תוכנת שידור חי חינמית ל-Windows ו-macOS. הורידו והריצו את ההתקנה — ללא הרשמה, ללא כרטיס, ללא סימן מים, ללא הגבלת זמן. זה המקודד ששולח את הווידאו שלכם ל-VirtWish.

2
הגדירו מצלמה וסצנה

פתחו SplitCam והוסיפו את המצלמה. בנו את הסצנה כפי שאתם רוצים שהצופים יראו — שכבות, טקסט, מצלמה שנייה או טלפון, מסנני יופי או רקע AI. הכול מוחל בזמן אמת לפני שהזרם עוזב את המחשב.

3
קבלו מפתח שידור של VirtWish

ב-VirtWish לחצו על האייקון בפינה הימנית העליונה → Profile → Start Broadcast, ואז Start Broadcast שוב כדי להגיע למקטע ה-OBS. העתיקו את הקישור בשורה הראשונה לשדה Stream URL של SplitCam, ואת ה-Stream Key הנפרד לשדה המפתח.

4
חברו את SplitCam ל-VirtWish

ב-SplitCam פתחו Stream Settings, הדביקו את URL השרת ומפתח השידור של VirtWish בשדות ה-RTMP המותאם. הגדירו ביטרייט 3,500–6,000 Kbps ל-1080p, 2,000–4,000 Kbps ל-720p, והריצו קודם את בדיקת המהירות המובנית.

5
לחצו Go Live

לחצו Go Live ב-SplitCam, ואז התחילו את השידור ב-VirtWish. תוך ~10 שניות המצלמה שלכם בלייב. השידורים הבאים בלחיצה אחת — פתחו SplitCam, Go Live.

טיפים מקצועיים

הקישור בשורה הראשונה

מקטע ה-OBS של VirtWish שם את ה-URL של הזרם בשורה הראשונה — העתיקו אותו ל-Stream URL של SplitCam, ואז את המפתח הנפרד.

שתי לחיצות על Start Broadcast

אתם לוחצים על Start Broadcast פעמיים ב-VirtWish כדי להגיע למקטע ה-OBS — זה נורמלי, לא תקלה.

השתמשו בחיבור קווי

Ethernet מנצח את ה-Wi-Fi בשידורים ארוכים — פריימים שאבדו פירושם טיפים שאבדו. חברו כבל למחשב המשדר.

עשו בדיקה פרטית קודם

הריצו שידור בדיקה קצר כדי לבדוק מצלמה, אודיו, מסגור ושכבות לפני שאתם פותחים את החדר לצופים.

פתרון בעיות

השידור ב-VirtWish נתקע או באפרינג

כמעט תמיד הביטרייט מוגדר גבוה ממה שההעלאה שלכם נושאת. הריצו את בדיקת המהירות המובנית של SplitCam, ואז הגדירו ביטרייט לכ-75% מההעלאה הנמדדת — 3,500–6,000 Kbps ל-1080p, נמוך יותר ל-720p. התקיעות נעלמת ברגע שהמקודד מפסיק לחרוג מהקו שלכם.

פריימים שאבדו במהלך שידור VirtWish

פריימים שאבדו פירושם שחבילות לא מגיעות ל-VirtWish בזמן — בדרך כלל Wi-Fi לא יציב. עברו ל-Ethernet קווי, סגרו אפליקציות שאוכלות רוחב פס והורידו מעט את הביטרייט. קפיצה בודדת לא נורא; עלייה מתמשכת פירושה שהקו לא עומד בקצב.

מסך שחור — הצופים לא רואים וידאו ב-VirtWish

המצלמה שלכם לא נבחרה כמקור פעיל ב-SplitCam, או שאפליקציה אחרת מחזיקה אותה. סגרו Zoom, Skype או OBS, בחרו שוב את המצלמה ברשימת המקורות של SplitCam וודאו שהתצוגה המקדימה מציגה את התמונה לפני לחיצה על Go Live.

VirtWish דוחה את מפתח השידור או לא מתחבר

העתיקו שוב את מפתח השידור — רווח בסוף או מפתח ישן שאופס הם הסיבות הנפוצות. ודאו ש-URL השרת תואם למה ש-VirtWish מציג וששידור במקודד חיצוני מופעל בחשבון. המחוון הירוק ב-Stream Settings של SplitCam מאשר מפתח תקף.

אין אודיו או אודיו לא מסונכרן ב-VirtWish

בחרו SplitCam גם כמצלמה וגם כמיקרופון, ובחרו את המיקרופון האמיתי שלכם בתוך מקור האודיו של SplitCam. אם האודיו מפגר אחרי הווידאו, הורידו את הרזולוציה ברמה אחת — המקודד עמוס מדי והאודיו מחכה לפריימים מאוחרים.

שאלות נפוצות

איפה נתוני ה-RTMP של VirtWish?

אייקון פינה ימנית עליונה → Profile → Start Broadcast פעמיים → מקטע ה-OBS מציג קישור ומפתח שידור.

האם VirtWish תומכת במקודד חיצוני?

כן — מקטע ה-OBS מספק URL זרם ומפתח, כך ש-SplitCam עובד.

איזה ביטרייט ל-VirtWish?

‏VirtWish לא מפרסמת תקרה רשמית; 3,500–6,000 Kbps ב-1080p בטוח. התאימו את רזולוציית SplitCam לזו שהוגדרה ב-VirtWish כדי למנוע שינוי גודל.

האם SplitCam חינמי ל-VirtWish?

כן — חינמי, ללא סימן מים וללא הגבלת זמן. הגדרת ה-URL-ומפתח של VirtWish לוקחת רק את כמה הדקות הנדרשות.
