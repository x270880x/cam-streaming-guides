视频指南

分步指南

1
下载并安装 SplitCam

SplitCam 是适用于 Windows 和 macOS 的免费直播软件。下载并运行安装程序——无需注册、无需信用卡、无水印、无时间限制。SplitCam 是将您的视频发送到 CamPlace 的编码器。

2
设置摄像头和场景

打开 SplitCam 并添加网络摄像头。按照您希望观众看到的方式构建场景——叠加层、文本、第二个摄像头或手机、美颜滤镜或 AI 背景。所有内容都在直播流离开 PC 之前实时应用。

3
获取您的 CamPlace 直播密钥

登录 CamPlace 并打开直播设置。切换到外部编码器 / RTMP / OBS 选项以显示服务器 URL 和直播密钥，复制两者。CamPlace 不发布官方 OBS 文档 — 上面的视频是通过当前界面最可靠的分步路径。

4
将 SplitCam 连接到 CamPlace

在 SplitCam 中打开 Stream Settings，将 CamPlace 服务器 URL 和直播密钥粘贴到自定义 RTMP 字段中。比特率：1080p 为 3,500–6,000 Kbps，720p 为 2,000–4,000 Kbps。先运行内置的速度测试。

5
点击 Go Live

在 SplitCam 中按 Go Live，然后在 CamPlace 上开始直播。约 10 秒内您的摄像头就会上线。后续直播只需一键——打开 SplitCam，点击 Go Live。

CamPlace 没有关于外部编码器的公开文章；上面的视频是当前路径的参考。

标准 RTMP 工作

虽然没有文档，CamPlace 接受标准 RTMP 服务器 URL 和密钥 — SplitCam 作为自定义 RTMP 目标连接。

使用有线连接

长时间直播时以太网比 Wi-Fi 更稳定 — 丢失的帧就是丢失的小费。把网线拉到直播电脑。

先做个私密测试

正式开播前运行一个简短的测试直播，检查摄像头、音频、构图和叠加层。

是 — 免费、无水印、无时间限制。由于 CamPlace 没有自己的编码器，免费的 RTMP 工具完成工作。
