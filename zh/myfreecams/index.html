快速回答
使用 SplitCam 在 MyFreeCams 直播：安装 SplitCam，构建场景，进入 Model Admin → Broadcaster，将 Web Broadcaster 切换为 External Broadcaster，复制服务器 URL 和 stream key，粘贴到 SplitCam，Go Live。

分步指南

1
下载并安装 SplitCam

SplitCam 是适用于 Windows 和 macOS 的免费直播编码器 — 无需注册、无需信用卡、无水印。

2
构建您的场景

在 SplitCam 中添加网络摄像头，叠加叠加层、文字、第二个摄像头或手机、美颜滤镜或 AI 背景。把 mfcalerts.com 作为 Browser 图层加到摄像头上方，让小费动画直接出现在 free chat 画面中。

3
在 Model Admin 切换到 External Broadcaster

登录 MFC，打开 Model Admin → Broadcaster，将 Web Broadcaster 切换为 External Broadcaster。复制显示的服务器 URL（rtmp://publish.myfreecams.com…）和 stream key。

4
在 SplitCam 中粘贴 URL 和 key

打开 SplitCam 的 Stream Settings，将 URL 和 stream key 粘贴到自定义 RTMP 字段。比特率设为 3,500–6,000 Kbps、1920×1080、30 fps、关键帧 2 秒。

5
点击 Go Live

在 SplitCam 按 Go Live。约 10 秒内信号就抵达 MFC，您在房间中正式上线。

专业提示

MFC 是纯小费/代币经济，不是订阅制

MFC 的核心不是订阅，而是 free chat 中的小费 — Premium 会员可以发起 private，但日常收入来自衣着状态下的 free chat。规划场景时要保证穿着随意时也能持续赚到小费。

Web Broadcaster vs External Broadcaster

默认的 Web Broadcaster 是单信号源工具；External Broadcaster 通过 SplitCam/OBS 解锁多场景、叠加层和美颜滤镜 — 这是高收入模特和新人之间的明显差距。

用 mfcalerts.com 做小费提醒

在 SplitCam 中把 mfcalerts.com 作为 Browser 图层添加到摄像头之上 — 直接在视频里显示小费动画和会员名称，会显著提高 free chat 的小费转化率。

使用有线连接

长时间直播时以太网比 Wi-Fi 更稳定 — 丢失的帧就是丢失的小费。把网线拉到直播电脑。

故障排除

您的 MyFreeCams 直播卡顿或缓冲

几乎总是比特率高于您的上传所能承受的。运行 SplitCam 内置的速度测试，然后将比特率设为测得上传的约 75% — 1080p 为 3,500–6,000 Kbps，720p 更低。一旦编码器不再超出您的连接，卡顿就会消失。

MyFreeCams 直播期间丢帧

丢帧意味着数据包未能及时到达 MyFreeCams — 通常是 Wi-Fi 不稳定。改用有线以太网连接，关闭占用带宽的应用，并稍微降低比特率。单次峰值没问题；持续上升说明连接跟不上。

黑屏 — 观众在 MyFreeCams 上看不到视频

您的摄像头未在 SplitCam 中被选为活动源，或被另一个应用占用。关闭 Zoom、Skype 或 OBS，在 SplitCam 的源列表中重新选择摄像头，并在按 Go Live 前确认预览显示您的画面。

MyFreeCams 拒绝直播密钥或无法连接

重新复制直播密钥 — 末尾多余的空格或旧的、已轮换的密钥是常见原因。确认服务器 URL 与 MyFreeCams 显示的一致，并且您的账户已启用外部编码器直播。SplitCam Stream Settings 中的绿色滑块确认密钥有效。

MyFreeCams 上没有声音或声音不同步

将 SplitCam 同时选为摄像头和麦克风，并在 SplitCam 的音频源中选择您的真实麦克风。如果音频落后于视频，将分辨率降低一档 — 编码器过载，音频在等待迟到的帧。

常见问题

MFC 支持 SplitCam 这类外部直播软件吗？

支持 — Model Admin 提供 External Broadcaster 选项，走标准 RTMP，OBS、SplitCam、vMix 都可以使用。

我的 MFC stream key 在哪里？

Model Admin → Broadcaster → External Broadcaster — 页面同时显示服务器 URL 和 stream key。两者都复制到 SplitCam。

MFC 用什么比特率？

MFC 支持最高约 6,000 Kbps、2 秒关键帧间隔。推 1920×1080、30 fps、3,500–6,000 Kbps 是稳妥配置。

SplitCam 在 MFC 上免费吗？

是 — SplitCam 完全免费，无水印，无时间限制。
