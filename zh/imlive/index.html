如何使用 SplitCam 与 ImLive

ImLive 直接在浏览器中使用网络摄像头 — 没有 RTMP 也没有直播密钥。免费 SplitCam 作为虚拟摄像头连接：在 SplitCam 中构建场景，然后在 ImLive 中将 SplitCam 选择为摄像头。

在 ImLive 上使用 SplitCam：安装 SplitCam，构建带媒体层的场景，打开 ImLive 并开始视频聊天，在 ImLive 设置中将 SplitCam 选择为网络摄像头和麦克风。

视频指南

分步指南

1
安装 SplitCam

SplitCam 是适用于 Windows 和 macOS 的免费软件。安装它 — 无水印、无注册。对于 ImLive，它作为虚拟摄像头工作，而不是 RTMP 编码器。

2
在 SplitCam 中构建场景

打开 SplitCam 并使用 Media Layers + 添加网络摄像头以及任何叠加层、文本、滤镜或 AI 背景。这个合成场景就是 ImLive 将看到的您的摄像头。

3
在 ImLive 上开始视频聊天

登录 ImLive 网站并点击 Start Video Chat，然后打开 Go To Settings 以访问摄像头和麦克风选项。

4
选择 SplitCam 作为摄像头

在 ImLive 设置中将 SplitCam 选择为网络摄像头和麦克风。ImLive 现在显示您的完整 SplitCam 场景，而不是平面的网络摄像头。

5
开始 Free Live Chat

在 ImLive 上点击 Free Live Chat 进入直播。要在会话中途更改外观，请在 SplitCam 中编辑场景 — ImLive 立即更新。

专业提示

无需直播密钥

ImLive 没有 RTMP — 不要寻找服务器 URL 或密钥。SplitCam 只是被选为摄像头设备。

将 SplitCam 也设置为麦克风

除了摄像头之外，将 SplitCam 选择为麦克风，以便您的音频混合和降噪也能进入直播。

在直播前构建场景

ImLive 显示 SplitCam 发送的内容 — 在开始聊天之前安排好图层。

先做个私密测试

正式开播前运行一个简短的测试直播，检查摄像头、音频、构图和叠加层。

常见问题

ImLive 使用 RTMP 还是直播密钥？

都不 — ImLive 通过浏览器直接使用网络摄像头。SplitCam 作为虚拟摄像头连接，没有要复制的密钥。

如何在 ImLive 中选择 SplitCam？

Start Video Chat → Go To Settings → 选择 SplitCam 作为网络摄像头和麦克风。

我可以在 ImLive 中使用叠加层吗？

是 — 在 SplitCam 场景中构建它们；ImLive 显示合成结果。

SplitCam 对 ImLive 免费吗？

是 — 免费、无水印、无时间限制。作为 ImLive 的虚拟摄像头，它不会给您的视频聊天添加成本或品牌。
