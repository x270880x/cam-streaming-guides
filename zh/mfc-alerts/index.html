如何将 MFC Alerts 添加到您的直播

MFC Alerts 在观众发送小费时在您的 MyFreeCams 直播中显示动画效果。它在免费 SplitCam 中作为浏览器层运行 — 设置一次，小费会实时触发屏幕上的反应。

快速回答
使用 SplitCam 添加 MFC Alerts：安装 SplitCam，添加网络摄像头，打开浏览器选项卡并加载 mfcalerts.com，复制您的 alerts URL，将其作为浏览器层添加在网络摄像头上方，然后用测试小费测试。

视频指南

分步指南

1
安装 SplitCam 并添加网络摄像头

为 Windows 或 macOS 安装免费 SplitCam，然后将网络摄像头作为源添加。MFC Alerts 作为此摄像头上方的层。

2
打开浏览器选项卡并转到 mfcalerts.com

在 SplitCam 中打开浏览器选项卡并导航到 www.mfcalerts.com。登录，或在还没有 mfcalerts.com 账户的情况下注册。

3
复制您的 alerts URL

在 mfcalerts.com 上使用 Copy to clipboard 复制您的个人 alerts URL — 这是渲染小费动画的页面。

4
将 URL 作为浏览器层添加 — 在网络摄像头上方

将 URL 粘贴到 SplitCam 浏览器窗口并点击 Add。然后重新排序源列表，使 MFC Alerts 在网络摄像头上方（3 点菜单 → Move Up）。如果它在下面，效果将保持隐藏。

5
用测试小费测试

打开 Settings → Send test tip 并确认 alert 效果出现在您的摄像头上方。然后正常在 MyFreeCams 上直播 — 真实的小费现在触发动画。

专业提示

MFC Alerts 必须在网络摄像头上方

最常见的错误：如果 MFC Alerts 层在源列表中的网络摄像头下方，效果将保持隐藏。向上移动。

需要 mfcalerts.com 账户

alerts URL 是个人的 — 如果您没有账户，请先在 mfcalerts.com 注册。

在直播前发送测试小费

使用 Settings → Send test tip 确认叠加层正在工作 — 不要在节目中途发现错误。

使用有线连接

长时间直播时以太网比 Wi-Fi 更稳定 — 丢失的帧就是丢失的小费。把网线拉到直播电脑。

常见问题

什么是 MFC Alerts？

MyFreeCams 的通知系统，在观众发送小费时在您的直播中显示视频效果 — 在 SplitCam 中作为浏览器叠加层添加。

为什么我的 MFC Alerts 不出现？

几乎总是层顺序 — MFC Alerts 浏览器层必须在 SplitCam 源列表中的网络摄像头上方。

我需要 MFC Alerts 的账户吗？

是 — 在 mfcalerts.com 注册以获取您的个人 alerts URL。

SplitCam 对此免费吗？

是 — SplitCam 免费、无水印、无时间限制，MFC Alerts 浏览器叠加层在其中运行而无需额外成本。
