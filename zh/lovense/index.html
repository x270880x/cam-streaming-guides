快速回答
要将 Lovense 玩具添加到直播：安装 SplitCam 和 Lovense 软件，配对玩具，将 Lovense 链接到您的 cam 平台，在 SplitCam 中将 Lovense 状态添加为浏览器层，然后正常直播。

视频指南

分步指南

1
安装 SplitCam

SplitCam 是适用于 Windows 和 macOS 的免费直播软件 — 将您的视频发送到 cam 平台的编码器。安装；无水印。

2
安装 Lovense 软件并配对玩具

安装 Lovense Connect / Lovense Stream（官方桌面应用）。打开玩具并通过蓝牙配对，使应用显示已连接。

3
将 Lovense 链接到您的 cam 平台

在 Lovense 应用中链接您的 cam 账户，使玩具对观众的代币/小费做出反应。大多数主要 cam 平台都受支持。

4
在 SplitCam 中添加 Lovense 叠加层

Lovense 提供叠加层/小部件 URL。在 SplitCam 场景中作为浏览器层添加，使观众在屏幕上看到玩具状态和最近的小费。

5
构建场景和 Go Live

添加摄像头和其他叠加层，将 cam 平台的 RTMP 密钥粘贴到 SplitCam 并点击 Go Live。玩具实时对小费做出反应。

专业提示

使用官方 Lovense SplitCam Toolset

Lovense 发布了带有自己安装指南的 SplitCam 特定工具集 — 它添加了为 SplitCam 制作的玩具活动叠加层和小费提醒。

更新 Lovense Cam Extension

工具集需要最新的 Lovense Cam Extension（30.1.4 或更新版本）— 在直播前更新。

保持玩具充电

节目中途几乎耗尽的电池会杀死互动方 — 在直播前充满电。

测试代币反应

发送一个小的测试小费以确认玩具在打开房间之前做出反应。

检查版本要求

Lovense SplitCam Toolset 需要 SplitCam 10.4.5 或更新版本。Lovense Cam Extension 官方涵盖 Chaturbate、Stripchat、BongaCams、MyFreeCams 和 CamSoda — 对于任何其他网站，使用 Lovense 的 Generic URL 集成。

常见问题

Lovense 官方支持 SplitCam 吗？

是 — Lovense 记录了带有自己安装指南的官方「Lovense SplitCam Toolset」，SplitCam 提供官方 Lovense 插件。集成从两方面都得到支持。

玩具直接连接到 SplitCam 吗？

不 — 玩具与 Lovense 应用配对；SplitCam 显示 Lovense 叠加层并直播摄像头。

哪些 cam 网站支持 Lovense？

Lovense Cam Extension 官方支持 Chaturbate、Stripchat、BongaCams、MyFreeCams 和 CamSoda，对其他的支持不同 — 在 Lovense 应用中查看当前列表。

我可以在屏幕上显示最近的小费吗？

是 — 在 SplitCam 中将 Lovense 小部件 URL 添加为浏览器层。
