如何使用 SplitCam 在 Streamen 直播

Streamen 是一家直播 cam 平台，模特向以打赏驱动的观众直播。它的直播设置开放标准的外部编码器通道，免费 SplitCam 接入这里 —— 让您用多机位场景、叠加层和滤镜直播，而不是一路平面网络摄像头。

快速回答
使用 SplitCam 在 Streamen 直播：安装 SplitCam，搭好场景，在模特后台打开 直播设置 → 外部编码器，复制服务器 URL 和 stream key，粘贴到 SplitCam，Go Live。

分步指南

1
下载并安装 SplitCam

SplitCam 是适用于 Windows 和 macOS 的免费直播软件 —— 无需注册、无需信用卡、无水印。

2
搭好场景

打开 SplitCam 添加摄像头。叠加叠加层、打赏目标文字、第二个摄像头或手机、美颜滤镜或 AI 背景。

3
获取 Streamen 的 URL 和 stream key

登录 Streamen 模特账号，打开直播设置 → 外部编码器，复制服务器 URL 和 stream key。

4
把 SplitCam 接到 Streamen

在 SplitCam 打开 Stream Settings，把 Streamen 的服务器 URL 和 stream key 粘贴到自定义 RTMP 字段。比特率设为 3,500–6,000 Kbps、1920×1080、30 fps、关键帧 2 秒。

5
点击 Go Live

在 SplitCam 按 Go Live，然后在后台上线。约 10 秒内信号就抵达 Streamen 的观众端。

专业提示

打赏驱动的观众

Streamen 的观众靠打赏 —— 屏幕上的打赏目标加上精致的场景，比一路平面网络摄像头更能催动打赏。

外部编码器解锁场景

走 SplitCam 的 RTMP 而不是基础的浏览器摄像头，才能启用多机位布局、叠加层和滤镜。

锁定您的分辨率

手动设为 1080p，免得画面悄悄掉质量；自适应直播流在静态画面下比特率下降是正常的。

使用有线连接

长时间直播时以太网比 Wi-Fi 更稳定 — 丢失的帧就是丢失的小费。把网线拉到直播电脑。

故障排除

您的 Streamen 直播卡顿或缓冲

几乎总是比特率高于您的上传所能承受的。运行 SplitCam 内置的速度测试，然后将比特率设为测得上传的约 75% — 1080p 为 3,500–6,000 Kbps，720p 更低。一旦编码器不再超出您的连接，卡顿就会消失。

Streamen 直播期间丢帧

丢帧意味着数据包未能及时到达 Streamen — 通常是 Wi-Fi 不稳定。改用有线以太网连接，关闭占用带宽的应用，并稍微降低比特率。单次峰值没问题；持续上升说明连接跟不上。

黑屏 — 观众在 Streamen 上看不到视频

您的摄像头未在 SplitCam 中被选为活动源，或被另一个应用占用。关闭 Zoom、Skype 或 OBS，在 SplitCam 的源列表中重新选择摄像头，并在按 Go Live 前确认预览显示您的画面。

Streamen 拒绝直播密钥或无法连接

重新复制直播密钥 — 末尾多余的空格或旧的、已轮换的密钥是常见原因。确认服务器 URL 与 Streamen 显示的一致，并且您的账户已启用外部编码器直播。SplitCam Stream Settings 中的绿色滑块确认密钥有效。

Streamen 上没有声音或声音不同步

将 SplitCam 同时选为摄像头和麦克风，并在 SplitCam 的音频源中选择您的真实麦克风。如果音频落后于视频，将分辨率降低一档 — 编码器过载，音频在等待迟到的帧。

常见问题

Streamen 支持 SplitCam 这种外部编码器吗？

支持 —— 直播设置开放标准的外部编码器 / RTMP 通道。完成认证后，把服务器 URL 和 stream key 复制到 SplitCam。

我的 Streamen stream key 在哪里？

在模特后台的直播 / 外部编码器设置里 —— 服务器 URL 和 stream key 都显示在那里。两者都粘贴到 SplitCam 的自定义 RTMP 字段。

Streamen 用什么比特率？

推 1920×1080、30 fps、3,500–6,000 Kbps、关键帧间隔 2 秒。先用 SplitCam 自带的速度测试跑一遍。

SplitCam 在 Streamen 上免费吗？

免费 —— SplitCam 完全免费、无水印、无时间限制。Streamen 在后台开放的外部编码器选项也免费。

模特在 Streamen 上能赚多少钱？

在 Streamen 上的收入取决于观众规模、直播时长和打赏情况。活跃的主播每月通常能拿到 $200–$3,000；顶级主播可达 $10,000 以上。你的分成比例遵循 Streamen 的佣金政策——开播前请仔细查看模特协议。

Streamen 对主播来说安全吗？

Streamen 在结算前会要求年龄和身份验证，这能保护模特免受欺诈。建议使用艺名，绝不在镜头前透露个人信息，开启地区屏蔽以隐藏所在地，并把每一位观众的请求都视为交易行为。SplitCam 的叠加层和 AI 背景也可以遮挡或替换你的真实环境。

在 Streamen 注册模特需要哪些证件？

Streamen 通常需要政府签发的带照片身份证件（护照、驾照或身份证）、手持证件的自拍，以及税务/结算表格（美国用户 W-9，非美国用户 W-8BEN）。审核一般需要 24–72 小时；通过后当天即可开播。

我可以用手机在 Streamen 上直播吗？

Streamen 通常提供手机主播 App 或移动网页版主播界面，但功能有限——没有叠加层、没有第二路摄像头、没有 AI 背景。如需完整的制作品质，请使用电脑配合 SplitCam 直播，并把手机作为第二路摄像头使用（SplitCam 支持来自手机的 IP 摄像头输入）。

Streamen 支持 OBS 或外部编码器吗？

支持——Streamen 会在主播面板中提供 RTMP 服务器地址和推流密钥。把它们粘贴到 SplitCam 的 Stream Settings → Custom RTMP，设置 1920×1080、30 fps、4,000–5,000 Kbps 码率，然后点击 Go Live。通过 Custom RTMP 通道，你可以充分利用 SplitCam 的场景合成（多摄像头、叠加层、滤镜）。
