快速回答
使用 SplitCam 在 Flirt4Free 直播：安装 SplitCam，构建场景，打开 Flirt4Free External Broadcast Form，复制 RTMP URL 和 Stream Name 并设置分辨率/比特率，粘贴到 SplitCam，Go Live。

视频指南

分步指南

1
下载并安装 SplitCam

SplitCam 是适用于 Windows 和 macOS 的免费直播软件。下载并运行安装程序——无需注册、无需信用卡、无水印、无时间限制。SplitCam 是将您的视频发送到 Flirt4Free 的编码器。

2
设置摄像头和场景

打开 SplitCam 并添加网络摄像头。按照您希望观众看到的方式构建场景——叠加层、文本、第二个摄像头或手机、美颜滤镜或 AI 背景。所有内容都在直播流离开 PC 之前实时应用。

3
获取您的 Flirt4Free 直播密钥

在 Flirt4Free 模特区域打开 External Broadcast Form。与大多数 cam 网站不同，Flirt4Free 给您一个 RTMP URL 和一个 Stream Name（不是「密钥」），加上您在表单本身中填写的分辨率和比特率字段。将 URL 和 Stream Name 复制到 SplitCam 服务器和密钥字段。

4
将 SplitCam 连接到 Flirt4Free

在 SplitCam 中打开 Stream Settings，将 Flirt4Free 服务器 URL 和直播密钥粘贴到自定义 RTMP 字段中。比特率：1080p 为 3,500–6,000 Kbps，720p 为 2,000–4,000 Kbps。先运行内置的速度测试。

5
点击 Go Live

在 SplitCam 中按 Go Live，然后在 Flirt4Free 上开始直播。约 10 秒内您的摄像头就会上线。后续直播只需一键——打开 SplitCam，点击 Go Live。

专业提示

是 Stream Name，不是密钥

Flirt4Free 称凭据为 Stream Name。您将它粘贴到 SplitCam 直播密钥字段 — 相同的作用。

在表单中设置分辨率/比特率

Flirt4Free 的 External Broadcast Form 有自己的分辨率和比特率字段 — 与 SplitCam 设置对齐，这样图像就不会被重新缩放。

历史平台

Flirt4Free 从 90 年代开始运行 — 模特工具成熟，External Broadcast Form 是其文档化的一部分。

使用有线连接

长时间直播时以太网比 Wi-Fi 更稳定 — 丢失的帧就是丢失的小费。把网线拉到直播电脑。

故障排除

您的 Flirt4Free 直播卡顿或缓冲

几乎总是比特率高于您的上传所能承受的。运行 SplitCam 内置的速度测试，然后将比特率设为测得上传的约 75% — 1080p 为 3,500–6,000 Kbps，720p 更低。一旦编码器不再超出您的连接，卡顿就会消失。

Flirt4Free 直播期间丢帧

丢帧意味着数据包未能及时到达 Flirt4Free — 通常是 Wi-Fi 不稳定。改用有线以太网连接，关闭占用带宽的应用，并稍微降低比特率。单次峰值没问题；持续上升说明连接跟不上。

黑屏 — 观众在 Flirt4Free 上看不到视频

您的摄像头未在 SplitCam 中被选为活动源，或被另一个应用占用。关闭 Zoom、Skype 或 OBS，在 SplitCam 的源列表中重新选择摄像头，并在按 Go Live 前确认预览显示您的画面。

Flirt4Free 拒绝直播密钥或无法连接

重新复制直播密钥 — 末尾多余的空格或旧的、已轮换的密钥是常见原因。确认服务器 URL 与 Flirt4Free 显示的一致，并且您的账户已启用外部编码器直播。SplitCam Stream Settings 中的绿色滑块确认密钥有效。

Flirt4Free 上没有声音或声音不同步

将 SplitCam 同时选为摄像头和麦克风，并在 SplitCam 的音频源中选择您的真实麦克风。如果音频落后于视频，将分辨率降低一档 — 编码器过载，音频在等待迟到的帧。

常见问题

Flirt4Free 支持外部编码器吗？

是 — 官方，通过 External Broadcast Form，它提供 RTMP URL 和 Stream Name。SplitCam 作为编码器工作。

我从哪里获取 Flirt4Free 的 RTMP 数据？

从模特区域的 External Broadcast Form — 显示 RTMP URL、Stream Name 和分辨率/比特率字段。

Flirt4Free 用什么比特率？

1080p 下 3,500–6,000 Kbps — 在 External Broadcast Form 和 SplitCam 中设置相同的值。

SplitCam 对 Flirt4Free 免费吗？

是 — 免费、无水印、无时间限制，这与像 Flirt4Free 这样节目可能很长的历史平台匹配。
