视频指南

分步指南

1
下载并安装 SplitCam

SplitCam 是适用于 Windows 和 macOS 的免费直播软件。下载并运行安装程序——无需注册、无需信用卡、无水印、无时间限制。SplitCam 是将您的视频发送到 BongaCams 的编码器。

2
设置摄像头和场景

打开 SplitCam 并添加网络摄像头。按照您希望观众看到的方式构建场景——叠加层、文本、第二个摄像头或手机、美颜滤镜或 AI 背景。所有内容都在直播流离开 PC 之前实时应用。

3
获取您的 BongaCams 直播密钥

在 BongaCams 打开 Options → Broadcast settings → Select Encoder → External Encoder，复制显示的服务器 URL 和直播密钥。如果 External Encoder 按钮缺失，联系 BongaCams 支持并请求在您的账户上启用外部编码。

4
将 SplitCam 连接到 BongaCams

在 SplitCam 中打开 Stream Settings，将 BongaCams 服务器 URL 和直播密钥粘贴到自定义 RTMP 字段中。比特率：1080p 为 3,500–6,000 Kbps，720p 为 2,000–4,000 Kbps。先运行内置的速度测试。

5
点击 Go Live

在 SplitCam 中按 Go Live，然后在 BongaCams 上开始直播。约 10 秒内您的摄像头就会上线。后续直播只需一键——打开 SplitCam，点击 Go Live。

专业提示

没有 External Encoder 按钮？支持

BongaCams 按账户启用外部编码 — 如果该选项在 Broadcast settings 中缺失，支持人员会激活。

匹配分辨率

BongaCams 建议网络摄像头分辨率和直播分辨率匹配 — 例如都是 1280×720。

使用有线连接

长时间直播时以太网比 Wi-Fi 更稳定 — 丢失的帧就是丢失的小费。把网线拉到直播电脑。

先做个私密测试

正式开播前运行一个简短的测试直播，检查摄像头、音频、构图和叠加层。

故障排除

您的 BongaCams 直播卡顿或缓冲

几乎总是比特率高于您的上传所能承受的。运行 SplitCam 内置的速度测试，然后将比特率设为测得上传的约 75% — 1080p 为 3,500–6,000 Kbps，720p 更低。一旦编码器不再超出您的连接，卡顿就会消失。

BongaCams 直播期间丢帧

丢帧意味着数据包未能及时到达 BongaCams — 通常是 Wi-Fi 不稳定。改用有线以太网连接，关闭占用带宽的应用，并稍微降低比特率。单次峰值没问题；持续上升说明连接跟不上。

黑屏 — 观众在 BongaCams 上看不到视频

您的摄像头未在 SplitCam 中被选为活动源，或被另一个应用占用。关闭 Zoom、Skype 或 OBS，在 SplitCam 的源列表中重新选择摄像头，并在按 Go Live 前确认预览显示您的画面。

BongaCams 拒绝直播密钥或无法连接

重新复制直播密钥 — 末尾多余的空格或旧的、已轮换的密钥是常见原因。确认服务器 URL 与 BongaCams 显示的一致，并且您的账户已启用外部编码器直播。SplitCam Stream Settings 中的绿色滑块确认密钥有效。

BongaCams 上没有声音或声音不同步

将 SplitCam 同时选为摄像头和麦克风，并在 SplitCam 的音频源中选择您的真实麦克风。如果音频落后于视频，将分辨率降低一档 — 编码器过载，音频在等待迟到的帧。

常见问题

为什么 External Encoder 按钮在 BongaCams 不显示？

外部编码在每个账户上不是默认激活 — 联系 BongaCams 支持激活它，按钮会出现在 Broadcast settings 中。

我需要验证我的 BongaCams 账户吗？

是 — 直播需要 18+、年龄验证的官方 ID 和上线前的账户批准。

BongaCams 用什么比特率？

BongaCams RTMP 摄取将视频比特率限制在约 6,000 Kbps，2 秒关键帧 — 1080p 下 3,500–6,000 是理想区间；先测试上传。

SplitCam 对 BongaCams 免费吗？

是 — 完全免费、无水印、无时间限制，所以编码器不会减少您在 BongaCams 的代币收入。

模特在 BongaCams 上能赚多少钱？

在 BongaCams 上的收入取决于观众规模、直播时长和打赏情况。活跃的主播每月通常能拿到 $200–$3,000；顶级主播可达 $10,000 以上。你的分成比例遵循 BongaCams 的佣金政策——开播前请仔细查看模特协议。

BongaCams 对主播来说安全吗？

BongaCams 在结算前会要求年龄和身份验证，这能保护模特免受欺诈。建议使用艺名，绝不在镜头前透露个人信息，开启地区屏蔽以隐藏所在地，并把每一位观众的请求都视为交易行为。SplitCam 的叠加层和 AI 背景也可以遮挡或替换你的真实环境。

在 BongaCams 注册模特需要哪些证件？

BongaCams 通常需要政府签发的带照片身份证件（护照、驾照或身份证）、手持证件的自拍，以及税务/结算表格（美国用户 W-9，非美国用户 W-8BEN）。审核一般需要 24–72 小时；通过后当天即可开播。

我可以用手机在 BongaCams 上直播吗？

BongaCams 通常提供手机主播 App 或移动网页版主播界面，但功能有限——没有叠加层、没有第二路摄像头、没有 AI 背景。如需完整的制作品质，请使用电脑配合 SplitCam 直播，并把手机作为第二路摄像头使用（SplitCam 支持来自手机的 IP 摄像头输入）。

BongaCams 支持 OBS 或外部编码器吗？

支持——BongaCams 会在主播面板中提供 RTMP 服务器地址和推流密钥。把它们粘贴到 SplitCam 的 Stream Settings → Custom RTMP，设置 1920×1080、30 fps、4,000–5,000 Kbps 码率，然后点击 Go Live。通过 Custom RTMP 通道，你可以充分利用 SplitCam 的场景合成（多摄像头、叠加层、滤镜）。
