SplitCamでCAM4配信する方法

CAM4は地域コントロール内蔵のグローバルcam-and-earnプラットフォーム — 選択した国で配信を隠せる。無料SplitCamを外部エンコーダーとして配信すると、基本ブロードキャスターでは不可能なシーン切替やオーバーレイが開く。

簡単な答え
SplitCamでCAM4配信：SplitCamをインストール、シーンを構築、CAM4でBroadcast & Earn Money → Start Broadcast → External Encoderへ、Get Stream Key、SplitCamに貼り付け、Go Live。

ビデオガイド

ステップバイステップ

1
SplitCamをダウンロードしてインストール

SplitCamはWindowsとmacOS用の無料ライブ配信ソフトウェアです。ダウンロードしてインストーラーを実行してください — 登録不要、カード不要、透かしなし、時間制限なし。SplitCamは、あなたのビデオをCAM4に送信するエンコーダーです。

2
カメラとシーンをセットアップ

SplitCamを開いてウェブカメラを追加します。視聴者に見せたいようにシーンを構築 — オーバーレイ、テキスト、2台目のカメラまたはスマートフォン、ビューティーフィルターやAI背景。すべてがストリームがPCを離れる前にライブで適用されます。

3
CAM4のストリームキーを取得

CAM4でBroadcast → Broadcast & Earn Money → Start Broadcast、上部のExternal Encoder。生年月日、性別、国を入力、Get Stream Keyでコピー。SplitCam Stream Settingsの緑のスライダーが接続を確認。

4
SplitCamをCAM4に接続

SplitCamでStream Settingsを開き、CAM4のサーバーURLとストリームキーをカスタムRTMPフィールドに貼り付けます。ビットレート：1080pは3,500–6,000 Kbps、720pは2,000–4,000 Kbps。最初に内蔵の速度テストを実行してください。

5
Go Liveをクリック

SplitCamでGo Liveを押してから、CAM4で配信を開始。約10秒以内にカメラがライブになります。次回以降の配信はワンクリック — SplitCamを開いてGo Live。

プロのヒント

地域制限を設定

CAM4は特定の国・地域で配信を隠せる — go-live前にCAM4側で設定。

緑のスライダーを見る

CAM4設定はキーが受け入れられるとSplitCam Stream Settingsに緑のスライダーを表示 — 赤 = キーをチェック。

通常より低いビットレート

CAM4のingestは動画ビットレートを約3,000 Kbpsに制限 — 大半のcamサイトより低い。より高く押さない。

有線接続を使用

長い配信ではEthernetがWi-Fiに勝つ — 失われたフレームは失われたチップ。配信PCまでケーブルを引きましょう。

トラブルシューティング

CAM4 の配信がカクつく・バッファリングする

ほぼ常にビットレートがアップロードの限界を超えています。SplitCam の内蔵速度テストを実行し、ビットレートを測定したアップロードの約75%に設定してください — 1080pで3,500–6,000 Kbps、720pはそれ以下。エンコーダーが接続を追い越さなくなればカクつきは消えます。

CAM4 配信中のフレーム落ち

フレーム落ちはパケットが CAM4 に間に合っていないこと — 通常は不安定なWi-Fiが原因です。有線Ethernet接続に切り替え、帯域を食うアプリを閉じ、ビットレートを少し下げてください。単発のスパイクは問題なし；継続的な増加は接続が追いつかないサインです。

黒い画面 — CAM4 で視聴者に映像が映らない

カメラが SplitCam でアクティブソースとして選択されていないか、別のアプリが使用中です。Zoom、Skype、OBS を閉じ、SplitCam のソースリストでウェブカメラを選び直し、Go Live を押す前にプレビューに映像が表示されることを確認してください。

CAM4 がストリームキーを拒否する・接続しない

ストリームキーをコピーし直してください — 末尾のスペースや古い、すでにローテートされたキーが通常の原因です。サーバーURLが CAM4 の表示と一致し、アカウントで外部エンコーダー配信が有効か確認してください。SplitCam の Stream Settings の緑のスライダーが有効なキーを確認します。

CAM4 で音声がない・音ズレする

SplitCam をカメラとマイクの両方に選び、SplitCam の音声ソースで実際のマイクを選択してください。音声が映像から遅れる場合は解像度を1段下げてください — エンコーダーが過負荷で、音声が遅れたフレームを待っています。

よくある質問

CAM4はOBS / 外部エンコーダーを公式サポート？

はい — CAM4はサポートサイトに公式OBS Guideがあり、最高の体験のためにExternal Encoderを推奨。SplitCamは同じRTMPルートを使用。

CAM4ストリームを地域ブロックできる？

はい — CAM4は特定の国で配信を隠す地域制限が内蔵。CAM4で設定、SplitCamではない。

CAM4のストリームキーは？

Broadcast → Broadcast & Earn Money → Start Broadcast → External Encoder → Get Stream Key。

CAM4のビットレートは？

平均より低い — CAM4 ingestは30 fps、1秒キーフレームで~3,000 Kbpsに制限。公式テーブルは~3,000を超えないことを推奨。
