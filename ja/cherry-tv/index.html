Cherry.tv

SplitCamでCherry.tv配信する方法

Cherry.tvは比較的新しく急成長中のcamプラットフォームで、Web3寄りの仕組み — 暗号通貨での出金が可能で、LiveJasminのような老舗ネットワークよりも参入障壁が低いのが特徴です。デフォルトはブラウザ配信ですが、Streamer Dashboardは標準のExternal Encoder経路も提供 — 無料のSplitCamはそこに接続します。

簡単な答え
SplitCamでCherry.tv配信：SplitCamをインストール、シーンを構築、Streamer Dashboard → Broadcast Settings → External Encoderを開き、URLとstream keyをコピー、SplitCamに貼り付け、Go Live。

ステップバイステップ

1
SplitCamをダウンロード・インストール

SplitCamはWindowsとmacOS向けの無料配信エンコーダー — 登録なし、カードなし、透かしなし。

2
シーンを構築

ウェブカメラとオーバーレイを追加 — Cherry.tvの視聴者は若くプラットフォーム慣れしているので、画面のデザイン感と空気感が重要です。カラフルなオーバーレイ、動くテキスト、AI背景を組み合わせましょう。

3
URLとstream keyを取得

Streamer Dashboard → Broadcast Settings → External EncoderでサーバーURLとstream keyをコピー。

4
SplitCamに貼り付け

SplitCamのStream Settingsを開き、URLとstream keyを貼り付け。ビットレートは3,500〜6,000 Kbps、1920×1080、30 fps、キーフレーム2秒に。

5
Go Liveをクリック

SplitCamでGo Liveを押した後、Streamer Dashboardでオンラインに切り替え — Cherry.tv上で即配信開始です。

プロのヒント

参入障壁が低く、トラフィックは伸び盛り

72時間のDocler式審査がない — Cherry.tvは早期参入で固定ファンを取りに行ける良いポジションです。

暗号通貨と法定通貨の両方で受け取れる

Cherry.tvは暗号通貨での出金にも対応 — 従来の決済が遅い／制限される地域では特に有利です。

ブラウザ配信 vs 外部エンコーダー

Cherry.tvのブラウザ配信は単一カメラのみ。SplitCamならマルチカメラシーン、オーバーレイ、美肌フィルター、AI背景まで解禁され、画面の作り込みで一段抜けます。

有線接続を使用

長い配信ではEthernetがWi-Fiに勝つ — 失われたフレームは失われたチップ。配信PCまでケーブルを引きましょう。

トラブルシューティング

Cherry.tv の配信がカクつく・バッファリングする

ほぼ常にビットレートがアップロードの限界を超えています。SplitCam の内蔵速度テストを実行し、ビットレートを測定したアップロードの約75%に設定してください — 1080pで3,500–6,000 Kbps、720pはそれ以下。エンコーダーが接続を追い越さなくなればカクつきは消えます。

Cherry.tv 配信中のフレーム落ち

フレーム落ちはパケットが Cherry.tv に間に合っていないこと — 通常は不安定なWi-Fiが原因です。有線Ethernet接続に切り替え、帯域を食うアプリを閉じ、ビットレートを少し下げてください。単発のスパイクは問題なし；継続的な増加は接続が追いつかないサインです。

黒い画面 — Cherry.tv で視聴者に映像が映らない

カメラが SplitCam でアクティブソースとして選択されていないか、別のアプリが使用中です。Zoom、Skype、OBS を閉じ、SplitCam のソースリストでウェブカメラを選び直し、Go Live を押す前にプレビューに映像が表示されることを確認してください。

Cherry.tv がストリームキーを拒否する・接続しない

ストリームキーをコピーし直してください — 末尾のスペースや古い、すでにローテートされたキーが通常の原因です。サーバーURLが Cherry.tv の表示と一致し、アカウントで外部エンコーダー配信が有効か確認してください。SplitCam の Stream Settings の緑のスライダーが有効なキーを確認します。

Cherry.tv で音声がない・音ズレする

SplitCam をカメラとマイクの両方に選び、SplitCam の音声ソースで実際のマイクを選択してください。音声が映像から遅れる場合は解像度を1段下げてください — エンコーダーが過負荷で、音声が遅れたフレームを待っています。

よくある質問

Cherry.tvは外部エンコーダーをサポートしている？

はい — Streamer DashboardのBroadcast Settings配下にExternal Encoderオプションがあり、標準RTMPです。OBS／SplitCamがそのまま使えます。

Cherry.tvのstream keyはどこ？

Streamer Dashboard → Broadcast Settings → External Encoder — URLとstream keyが同じページに表示されます。

Cherry.tvのビットレートは？

1920×1080、30 fps、3,500〜6,000 Kbps、キーフレーム2秒を狙ってください。

Cherry.tvでSplitCamは無料？

はい — SplitCamは完全無料、透かしなし、時間制限なし。

Cherry.tvでモデルはどれくらい稼げますか？

Cherry.tvでの収入は視聴者数、配信時間、投げ銭の状況によって変動します。アクティブな配信者の月収は通常2万〜30万円程度で、トップ層は100万円以上に達します。あなたの取り分はCherry.tvの手数料体系に従うため、配信開始前にモデル契約書を必ず確認してください。

Cherry.tvは配信者にとって安全ですか？

Cherry.tvは出金前に年齢と本人確認を必須としており、これがモデルを詐欺から守ります。ステージネームを使い、カメラの前で個人情報を絶対に明かさず、ジオブロック機能で自宅地域から配信を隠し、視聴者からのリクエストはすべて取引として扱ってください。SplitCamのオーバーレイやAI背景機能を使えば、実際の周囲の様子を隠したり差し替えたりすることもできます。

Cherry.tvでモデル登録するにはどんな書類が必要ですか？

Cherry.tvでは通常、政府発行の写真付き身分証（パスポート、運転免許証、マイナンバーカードなど）、その身分証を持った自撮り写真、そして税務・支払い用フォーム（米国居住者はW-9、それ以外はW-8BEN）が必要です。審査は通常24〜72時間で完了し、承認されればその日のうちに配信を開始できます。

Cherry.tvにスマホから配信できますか？

Cherry.tvには大抵モバイル配信アプリかモバイルウェブ配信ツールが用意されていますが、オーバーレイなし、セカンドカメラなし、AI背景なしと機能が制限されます。本格的な配信品質を求めるなら、パソコンからSplitCamで配信し、スマホはセカンドカメラとして使うのがおすすめです（SplitCamはスマホからのIPカメラ入力に対応しています）。

Cherry.tvはOBSや外部エンコーダーに対応していますか？

はい、対応しています。Cherry.tvの配信者パネルにRTMPサーバーURLとストリームキーが表示されます。両方をSplitCamの配信設定 → カスタムRTMPに貼り付け、1920×1080の30fps、ビットレート4,000〜5,000Kbpsに設定して配信開始をクリックしてください。カスタムRTMP経由ならSplitCamのシーン合成機能（マルチカメラ、オーバーレイ、フィルター）をフル活用できます。
