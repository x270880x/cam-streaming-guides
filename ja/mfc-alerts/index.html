MFC Alertsをストリームに追加する方法

MFC Alertsは視聴者がチップを送るたびにMyFreeCamsストリームにアニメーション効果を表示。無料SplitCam内でBrowserレイヤーとして動作 — 一度設定すれば、チップが画面上の反応をライブでトリガー。

簡単な答え
SplitCamでMFC Alertsを追加：SplitCamをインストール、ウェブカメラを追加、Browserタブを開いてmfcalerts.comをロード、アラートURLをコピー、ウェブカメラの上にBrowserレイヤーとして追加、テストチップでテスト。

ビデオガイド

ステップバイステップ

1
SplitCamをインストールしてウェブカメラを追加

WindowsまたはmacOS用の無料SplitCamをインストール、その後ウェブカメラをソースとして追加。MFC Alertsはこのカメラの上のレイヤーとして座る。

2
Browserタブを開いてmfcalerts.comに移動

SplitCamでBrowserタブを開きwww.mfcalerts.comに移動。ログインまたはmfcalerts.comアカウントをまだ持っていない場合は登録。

3
アラートURLをコピー

mfcalerts.comでCopy to clipboardを使って個人のアラートURLをコピー — それはチップアニメーションをレンダリングするページ。

4
URLをBrowserレイヤーとして追加 — ウェブカメラの上

URLをSplitCam Browserウィンドウに貼り付けてAddをクリック。その後ソースリストを並び替えてMFC Alertsをウェブカメラの上に（3点メニュー → Move Up）。下にあると効果は隠れたまま。

5
テストチップでテスト

Settings → Send test tipを開いてアラート効果がカメラの上に表示されることを確認。その後MyFreeCamsで通常通り配信 — 実際のチップが今アニメーションをトリガー。

プロのヒント

MFC Alertsはウェブカメラの上にある必要

最も一般的なエラー：MFC Alertsレイヤーがソースリストでウェブカメラの下にあると、効果は隠れたまま。上に移動。

mfcalerts.comアカウントが必要

アラートURLは個人的 — アカウントがない場合は最初にmfcalerts.comに登録。

ライブ前にテストチップを送信

Settings → Send test tipを使用してオーバーレイが動作することを確認 — ショーの途中でエラーを発見しない。

有線接続を使用

長い配信ではEthernetがWi-Fiに勝つ — 失われたフレームは失われたチップ。配信PCまでケーブルを引きましょう。

トラブルシューティング

MFC Alerts の配信がカクつく・バッファリングする

ほぼ常にビットレートがアップロードの限界を超えています。SplitCam の内蔵速度テストを実行し、ビットレートを測定したアップロードの約75%に設定してください — 1080pで3,500–6,000 Kbps、720pはそれ以下。エンコーダーが接続を追い越さなくなればカクつきは消えます。

MFC Alerts 配信中のフレーム落ち

フレーム落ちはパケットが MFC Alerts に間に合っていないこと — 通常は不安定なWi-Fiが原因です。有線Ethernet接続に切り替え、帯域を食うアプリを閉じ、ビットレートを少し下げてください。単発のスパイクは問題なし；継続的な増加は接続が追いつかないサインです。

黒い画面 — MFC Alerts で視聴者に映像が映らない

カメラが SplitCam でアクティブソースとして選択されていないか、別のアプリが使用中です。Zoom、Skype、OBS を閉じ、SplitCam のソースリストでウェブカメラを選び直し、Go Live を押す前にプレビューに映像が表示されることを確認してください。

MFC Alerts がストリームキーを拒否する・接続しない

ストリームキーをコピーし直してください — 末尾のスペースや古い、すでにローテートされたキーが通常の原因です。サーバーURLが MFC Alerts の表示と一致し、アカウントで外部エンコーダー配信が有効か確認してください。SplitCam の Stream Settings の緑のスライダーが有効なキーを確認します。

MFC Alerts で音声がない・音ズレする

SplitCam をカメラとマイクの両方に選び、SplitCam の音声ソースで実際のマイクを選択してください。音声が映像から遅れる場合は解像度を1段下げてください — エンコーダーが過負荷で、音声が遅れたフレームを待っています。

よくある質問

MFC Alertsとは？

視聴者がチップを送るときにストリームにビデオ効果を表示するMyFreeCams用通知システム — SplitCamにBrowserオーバーレイとして追加。

MFC Alertsが表示されないのはなぜ？

ほぼ常にレイヤー順序 — MFC Alerts BrowserレイヤーはSplitCamソースリストでウェブカメラの上にある必要。

MFC Alertsにアカウントが必要？

はい — mfcalerts.comに登録して個人のアラートURLを取得。

SplitCamはこれで無料？

はい — SplitCamは無料、透かしなし、時間制限なし、MFC Alerts browserオーバーレイは追加コストなしで内部で実行。
