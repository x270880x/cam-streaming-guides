Jerkmate

SplitCamでJerkmate配信する方法

Jerkmateは検索数の多いcamブランドの一つで、視聴者とライブパフォーマーをマッチングするAIマッチメーカーで知られています。ただしJerkmate自身はブロードキャスターを持たず、ライブモデルをStreamateネットワークから供給しています。あなたはStreamateネットワークのモデルとして配信し、そのショーがJerkmateとパートナーサイトへ配信されます。StreamateはSplitCamのチャンネルリストに事前構成された宛先として用意されているため、無料のSplitCamなら手動のRTMP入力なしで、マルチカメラシーン、オーバーレイ、フィルターを追加できます。

簡単な答え
SplitCamでJerkmate配信：SplitCamをインストール、シーンを構築、Jerkmateに供給するStreamateネットワークにモデルとして登録、SM Connect → Start Showでキーをコピー、その後SplitCamでStream Settings → Add Channel → Streamateを開いて貼り付け、Go Live。

ステップバイステップ

1
SplitCamをダウンロード・インストール

SplitCamはWindowsとmacOS向けの無料配信ソフト — 登録なし、カードなし、透かしなし。

2
シーンを構築

SplitCamでウェブカメラを追加し、オーバーレイ、テキスト、サブカメラやスマホ、美肌フィルター、AI背景を重ねます — 作り込んだシーンはJerkmateのマッチ済み視聴者を転換させます。

3
モデル登録してキーを取得

Jerkmateのモデルプログラム、または供給元のStreamateネットワークで直接登録します。SM Connectを開いて規約を承認、Start Showをクリックしてストリーミングキーをコピー。

4
SplitCamでStreamateをチャンネル追加

Stream Settings → Add Channelを開き、統合リストからStreamateを選んでキーを貼り付け — 手動RTMP URLは不要です。解像度は1080pに固定。

5
Go Liveをクリック

SplitCamでGo Liveを押すと、緑のスライダーが接続を確認します。ショーはStreamateネットワーク全体へ流れ、Jerkmateに表示されます。

プロのヒント

土台はStreamateネットワーク

Jerkmate専用のブロードキャスターを探さないでください — 配信先はStreamateで、Jerkmateがそれを再配信します。Streamateで使えるもの（SM Connect、SplitCam統合チャンネル）はそのままJerkmateでも使えます。

AIマッチング経由のトラフィックを取り込む

Jerkmateのマッチメーカーは、視聴者の好みに合うモデルへ誘導します — オーバーレイと適切な構図で磨いたSplitCamのシーンは、フラットなウェブカメラよりこのマッチ済み視聴者をはるかに高く転換します。

1本の配信で多サイトへ

Streamateネットワークへ配信すれば、Jerkmateとパートナーサイトへ同時に露出されます — 1本のSplitCam配信で追加設定なしにリーチを広げられます。

有線接続を使用

長い配信ではEthernetがWi-Fiに勝つ — 失われたフレームは失われたチップ。配信PCまでケーブルを引きましょう。

トラブルシューティング

Jerkmate の配信がカクつく・バッファリングする

ほぼ常にビットレートがアップロードの限界を超えています。SplitCam の内蔵速度テストを実行し、ビットレートを測定したアップロードの約75%に設定してください — 1080pで3,500–6,000 Kbps、720pはそれ以下。エンコーダーが接続を追い越さなくなればカクつきは消えます。

Jerkmate 配信中のフレーム落ち

フレーム落ちはパケットが Jerkmate に間に合っていないこと — 通常は不安定なWi-Fiが原因です。有線Ethernet接続に切り替え、帯域を食うアプリを閉じ、ビットレートを少し下げてください。単発のスパイクは問題なし；継続的な増加は接続が追いつかないサインです。

黒い画面 — Jerkmate で視聴者に映像が映らない

カメラが SplitCam でアクティブソースとして選択されていないか、別のアプリが使用中です。Zoom、Skype、OBS を閉じ、SplitCam のソースリストでウェブカメラを選び直し、Go Live を押す前にプレビューに映像が表示されることを確認してください。

Jerkmate がストリームキーを拒否する・接続しない

ストリームキーをコピーし直してください — 末尾のスペースや古い、すでにローテートされたキーが通常の原因です。サーバーURLが Jerkmate の表示と一致し、アカウントで外部エンコーダー配信が有効か確認してください。SplitCam の Stream Settings の緑のスライダーが有効なキーを確認します。

Jerkmate で音声がない・音ズレする

SplitCam をカメラとマイクの両方に選び、SplitCam の音声ソースで実際のマイクを選択してください。音声が映像から遅れる場合は解像度を1段下げてください — エンコーダーが過負荷で、音声が遅れたフレームを待っています。

よくある質問

Jerkmateに独自のブロードキャスターやストリームキーはありますか？

いいえ — JerkmateはStreamateネットワークからライブモデルを供給しています。SM Connect経由でStreamateネットワークのモデルとして配信すれば、ショーは自動的にJerkmateに表示されます。

Jerkmateのストリームキーはどこで取得しますか？

Streamateネットワークのモデル側でSM Connectから：規約を承認 → Start Show → キーをコピー。SplitCam統合のStreamateチャンネルに貼り付けるだけで、手動のRTMP URLは不要です。

Jerkmateのビットレートは？

1920×1080、30 fps、3,500〜6,000 Kbps、キーフレーム間隔2秒。ネットワーク配信は適応的なので、静止画でビットレートが下がるのは正常です。先にSplitCamのスピードテストを実行し、有線接続を使ってください。

JerkmateでSplitCamは無料？

はい — SplitCamは完全無料、透かしなし、時間制限なし。Jerkmateに供給するStreamateはSplitCam統合チャンネルなので、別途エンコーダーのコストもかかりません。

Jerkmateでモデルはどれくらい稼げますか？

Jerkmateでの収入は視聴者数、配信時間、投げ銭の状況によって変動します。アクティブな配信者の月収は通常2万〜30万円程度で、トップ層は100万円以上に達します。あなたの取り分はJerkmateの手数料体系に従うため、配信開始前にモデル契約書を必ず確認してください。

Jerkmateは配信者にとって安全ですか？

Jerkmateは出金前に年齢と本人確認を必須としており、これがモデルを詐欺から守ります。ステージネームを使い、カメラの前で個人情報を絶対に明かさず、ジオブロック機能で自宅地域から配信を隠し、視聴者からのリクエストはすべて取引として扱ってください。SplitCamのオーバーレイやAI背景機能を使えば、実際の周囲の様子を隠したり差し替えたりすることもできます。

Jerkmateでモデル登録するにはどんな書類が必要ですか？

Jerkmateでは通常、政府発行の写真付き身分証（パスポート、運転免許証、マイナンバーカードなど）、その身分証を持った自撮り写真、そして税務・支払い用フォーム（米国居住者はW-9、それ以外はW-8BEN）が必要です。審査は通常24〜72時間で完了し、承認されればその日のうちに配信を開始できます。

Jerkmateにスマホから配信できますか？

Jerkmateには大抵モバイル配信アプリかモバイルウェブ配信ツールが用意されていますが、オーバーレイなし、セカンドカメラなし、AI背景なしと機能が制限されます。本格的な配信品質を求めるなら、パソコンからSplitCamで配信し、スマホはセカンドカメラとして使うのがおすすめです（SplitCamはスマホからのIPカメラ入力に対応しています）。

JerkmateはOBSや外部エンコーダーに対応していますか？

はい、対応しています。Jerkmateの配信者パネルにRTMPサーバーURLとストリームキーが表示されます。両方をSplitCamの配信設定 → カスタムRTMPに貼り付け、1920×1080の30fps、ビットレート4,000〜5,000Kbpsに設定して配信開始をクリックしてください。カスタムRTMP経由ならSplitCamのシーン合成機能（マルチカメラ、オーバーレイ、フィルター）をフル活用できます。
