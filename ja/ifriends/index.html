iFriends

SplitCamでiFriends配信する方法

iFriends（WebPower）は最も長く続く独立系camネットワークの一つ — 派手さはないものの堅実に利益を上げ、固定ファンが厚く、ブラウザ配信が一般化する前から作り込まれた成熟したModel Centerを備えています。Model Centerからの標準的な外部エンコーダー経路に対応しており、無料のSplitCamはそこに接続 — この老舗ネットワーク上で、最新のマルチカメラシーン、オーバーレイ、フィルターを使って配信できます。

簡単な答え
SplitCamでiFriends配信：SplitCamをインストール、シーンを構築、Model CenterのBroadcast Settings → External Encoderを開く、サーバーURLとstream keyをコピー、SplitCamに貼り付け、Go Live。

ステップバイステップ

1
SplitCamをダウンロード・インストール

SplitCamはWindowsとmacOS向けの無料配信ソフト — 登録なし、カードなし、透かしなし。

2
シーンを構築

SplitCamでウェブカメラを追加し、オーバーレイ、テキスト、サブカメラやスマホ、美肌フィルター、AI背景を重ねます — 磨き上げた最新のシーンは、この老舗ネットワークでひときわ目立ちます。

3
iFriendsのURLとstream keyを取得

iFriendsのモデルアカウントにログインし、Model Centerを開いてBroadcast Settings → External Encoderへ。サーバーURLと固有のstream keyが表示されます。両方コピー。

4
SplitCamをiFriendsへ接続

SplitCamのStream Settingsを開き、iFriendsのサーバーURLとstream keyをカスタムRTMPフィールドに貼り付け。ビットレートは3,500〜6,000 Kbps、1920×1080、30 fps、キーフレーム2秒。

5
Go Liveをクリック

SplitCamでGo Liveを押し、iFriendsのModel Centerからオンラインに切り替え。約10秒以内に映像がネットワークへ届きます。

プロのヒント

ロングテールな常連層、成熟したネットワーク

iFriendsは90年代後半から続いており、ファン層も厚め — 多くが一見客ではなく長年のサブスクライバーです。確立したモデルには安定収入、新規には伸びがゆるやかな環境です。

ブラウザ配信 vs 外部エンコーダー

iFriendsの旧来のブラウザ配信は、最新のマルチカメラUXが登場する前に作られたもの。External Encoder経由でSplitCamに切り替えると体感がはっきり変わります — 旧ツールでは出せないマルチカメラシーン、オーバーレイ、美肌フィルターが使えます。

安定した支払い、サプライズが少ない

iFriendsの親会社（WebPower）は何十年もモデルへ確実に支払ってきました — 新興のクリプト対応プラットフォームより支払いサイクルは遅めですが、トラブルの話はほとんど聞きません。

有線接続を使用

長い配信ではEthernetがWi-Fiに勝つ — 失われたフレームは失われたチップ。配信PCまでケーブルを引きましょう。

トラブルシューティング

iFriends の配信がカクつく・バッファリングする

ほぼ常にビットレートがアップロードの限界を超えています。SplitCam の内蔵速度テストを実行し、ビットレートを測定したアップロードの約75%に設定してください — 1080pで3,500–6,000 Kbps、720pはそれ以下。エンコーダーが接続を追い越さなくなればカクつきは消えます。

iFriends 配信中のフレーム落ち

フレーム落ちはパケットが iFriends に間に合っていないこと — 通常は不安定なWi-Fiが原因です。有線Ethernet接続に切り替え、帯域を食うアプリを閉じ、ビットレートを少し下げてください。単発のスパイクは問題なし；継続的な増加は接続が追いつかないサインです。

黒い画面 — iFriends で視聴者に映像が映らない

カメラが SplitCam でアクティブソースとして選択されていないか、別のアプリが使用中です。Zoom、Skype、OBS を閉じ、SplitCam のソースリストでウェブカメラを選び直し、Go Live を押す前にプレビューに映像が表示されることを確認してください。

iFriends がストリームキーを拒否する・接続しない

ストリームキーをコピーし直してください — 末尾のスペースや古い、すでにローテートされたキーが通常の原因です。サーバーURLが iFriends の表示と一致し、アカウントで外部エンコーダー配信が有効か確認してください。SplitCam の Stream Settings の緑のスライダーが有効なキーを確認します。

iFriends で音声がない・音ズレする

SplitCam をカメラとマイクの両方に選び、SplitCam の音声ソースで実際のマイクを選択してください。音声が映像から遅れる場合は解像度を1段下げてください — エンコーダーが過負荷で、音声が遅れたフレームを待っています。

よくある質問

iFriendsはSplitCamのような外部エンコーダーを公式サポート？

はい — Model CenterのBroadcast Settings配下にExternal Encoderオプションがあります。標準RTMPのサーバーURLとstream keyで、アカウントが承認されればOBS、SplitCam、vMixがそのまま接続できます。

iFriendsのstream keyはどこ？

Model Center → Broadcast Settings → External Encoder。サーバーURLとstream keyが同じページに表示されます — SplitCamのカスタムRTMPフィールドに両方コピー。

iFriendsのビットレートは？

1920×1080、30 fps、3,500〜6,000 Kbps、キーフレーム間隔2秒を狙ってください。先にSplitCam内蔵のスピードテストを実行。

iFriendsでSplitCamは無料？

はい — SplitCamは完全無料、透かしなし、時間制限なし。iFriendsのExternal EncoderオプションもModel Center内で無料です。
