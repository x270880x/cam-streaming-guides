Hakbang-hakbang

1
I-install ang SplitCam

Ang SplitCam ay libreng streaming software para sa Windows at macOS — ang encoder na nagpapadala ng video mo sa cam platform. I-install; walang watermark.

2
I-install ang Lovense software at i-pair ang toy

I-install ang Lovense Connect / Lovense Stream (opisyal na desktop app). I-on ang toy at i-pair sa pamamagitan ng Bluetooth para magpakita ang app na connected.

3
I-link ang Lovense sa cam platform

Sa Lovense app i-link ang cam account para tumugon ang toy sa viewer tokens / tips. Sinusuportahan ang karamihan ng major cam platforms.

4
Idagdag ang Lovense overlay sa SplitCam

Nagbibigay ang Lovense ng overlay / widget URL. Idagdag bilang Browser layer sa SplitCam scene para makita ng viewers ang toy status at recent tips on screen.

5
Buuin ang scene at Go Live

Idagdag ang camera at iba pang overlays, i-paste ang cam platform RTMP key sa SplitCam at i-click ang Go Live. Tumutugon ang toy sa tips in real time.

Mga tip

Naglalathala ang Lovense ng SplitCam-specific toolset na may sariling installation guide — nagdadagdag ng toy activity overlay at tip alerts na ginawa para sa SplitCam.

I-update ang Lovense Cam Extension

Nangangailangan ang toolset ng recent Lovense Cam Extension (30.1.4 o mas bago) — i-update bago mag-live.

Panatilihing naka-charge ang toy

Halos walang baterya sa kalagitnaan ng show ay pumapatay sa interactive side — i-charge nang buo bago mag-live.

Subukan ang reaction sa tokens

Magpadala ng maliit na test tip para kumpirmahin na tumutugon ang toy bago buksan ang room.

Suriin ang version requirements

Nangangailangan ng SplitCam 10.4.5 o mas bago ang Lovense SplitCam Toolset. Sinasaklaw ng Lovense Cam Extension ang Chaturbate, Stripchat, BongaCams, MyFreeCams at CamSoda nang opisyal — para sa anumang ibang site, gamitin ang Generic URL integration ng Lovense.

Pag-troubleshoot

Ang Lovense stream mo ay nag-lalag o nag-bubuffer

Halos palaging mas mataas ang bitrate kaysa kaya ng upload mo. Patakbuhin ang built-in speed test ng SplitCam, tapos i-set ang bitrate sa mga 75% ng nasukat mong upload — 3,500–6,000 Kbps para sa 1080p, mas mababa para sa 720p. Nawawala ang lag kapag tumigil na ang encoder sa pag-unahan sa koneksyon mo.

Dropped frames habang nag-bo-broadcast sa Lovense

Ang dropped frames ay nangangahulugang hindi umaabot ang mga packet sa Lovense sa tamang oras — kadalasan ay hindi stable na Wi-Fi. Lumipat sa wired Ethernet connection, isara ang mga app na kumakain ng bandwidth, at bahagyang ibaba ang bitrate. Ok lang ang isang spike; ang tuloy-tuloy na pagtaas ay nangangahulugang hindi kaya ng koneksyon.

Black screen — walang nakikitang video ang viewers sa Lovense

Hindi napili ang camera mo bilang active source sa SplitCam, o may ibang app na humahawak dito. Isara ang Zoom, Skype o OBS, piliin muli ang webcam sa source list ng SplitCam, at tiyakin na ipinapakita ng preview ang feed mo bago pindutin ang Go Live.

Tinatanggihan ng Lovense ang stream key o ayaw kumonekta

Kopyahin muli ang stream key — ang trailing space o luma nang key na na-rotate na ang karaniwang dahilan. Tiyakin na tumutugma ang server URL sa ipinapakita ng Lovense at na naka-enable ang external-encoder broadcasting sa account mo. Isang green slider sa Stream Settings ng SplitCam ang nagkukumpirma ng valid na key.

Walang audio o hindi sync ang audio sa Lovense

Piliin ang SplitCam bilang camera AT microphone, at piliin ang totoong mic mo sa loob ng audio source ng SplitCam. Kung nahuhuli ang audio sa video, ibaba ang resolution ng isang hakbang — overloaded ang encoder at naghihintay ang audio sa mga late frame.

FAQ

Direktang kumokonekta ba ang toy sa SplitCam?

Hindi — ipinapares ang toy sa Lovense app; ipinapakita ng SplitCam ang Lovense overlay at nag-bo-broadcast ng camera.

Anong cam sites ang sumusuporta sa Lovense?

Maaari ko bang ipakita ang mga kamakailang tips sa screen?

Oo — idagdag ang Lovense widget URL bilang Browser layer sa SplitCam.
