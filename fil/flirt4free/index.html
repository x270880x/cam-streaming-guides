Hakbang-hakbang

1
I-download at i-install ang SplitCam

Ang SplitCam ay libreng live-streaming software para sa Windows at macOS. I-download ito at patakbuhin ang installer — walang sign-up, walang card, walang watermark, walang time limit. Ito ang encoder na nagpapadala ng iyong video sa Flirt4Free.

2
I-set up ang iyong camera at scene

Buksan ang SplitCam at idagdag ang iyong webcam. Buuin ang scene sa paraang gusto mong makita ng mga viewer — overlays, text, pangalawang camera o telepono, beauty filters o AI background. Ang lahat ay inilalapat nang live bago umalis ang stream sa iyong PC.

3
Kunin ang iyong Flirt4Free stream key

Sa Flirt4Free model area buksan ang External Broadcast Form. Hindi tulad ng karamihan ng cam sites, nagbibigay ang Flirt4Free ng RTMP URL at Stream Name (hindi «key»), at saka resolution at bitrate fields na pinupunan mo sa form mismo. Kopyahin ang URL at Stream Name sa server at key fields ng SplitCam.

4
Ikonekta ang SplitCam sa Flirt4Free

Sa SplitCam buksan ang Stream Settings, i-paste ang Flirt4Free server URL at stream key sa mga custom RTMP field. Bitrate: 3,500–6,000 Kbps para sa 1080p, 2,000–4,000 Kbps para sa 720p. Patakbuhin muna ang built-in speed test.

5
I-click ang Go Live

Pindutin ang Go Live sa SplitCam, pagkatapos ay simulan ang broadcast sa Flirt4Free. Sa loob ng ~10 segundo, live na ang iyong camera. Ang mga susunod na stream ay isang click lang — buksan ang SplitCam, Go Live.

Mga tip

Stream Name ito, hindi key

Tinatawag ng Flirt4Free ang credential bilang Stream Name. I-paste mo ito sa SplitCam stream key field — parehong tungkulin.

I-set ang resolution/bitrate sa form

Ang External Broadcast Form ng Flirt4Free ay may sariling resolution at bitrate fields — i-align sa SplitCam settings para hindi ma-rescale ang image.

Historikal na platform

Tumatakbo ang Flirt4Free simula noong 90s — mature ang model tools, at ang External Broadcast Form ay documented bahagi nito.

Gamitin ang wired connection

Talo ng Ethernet ang Wi-Fi sa mahabang stream — ang naturang frame ay nawawalang tip. Maglagay ng cable papunta sa stream PC.

FAQ

Sinusuportahan ba ng Flirt4Free ang external encoders?

Oo — opisyal, sa pamamagitan ng External Broadcast Form, na nagbibigay ng RTMP URL at Stream Name. Gumagana ang SplitCam bilang encoder.

Saan ako kukuha ng Flirt4Free RTMP data?

Mula sa External Broadcast Form sa model area — nagpapakita ng RTMP URL, Stream Name at resolution/bitrate fields.

Anong bitrate para sa Flirt4Free?

3,500–6,000 Kbps sa 1080p — i-set ang parehong halaga sa External Broadcast Form at sa SplitCam.

Libre ba ang SplitCam para sa Flirt4Free?

Oo — libre, walang watermark at walang time limit, na bumabagay sa historikal na platform tulad ng Flirt4Free kung saan ang shows ay maaaring mahaba.
