Как вести трансляцию на Streamate через SplitCam

Streamate — устоявшаяся кам-платформа, и это одна из площадок, предустановленных в SplitCam в списке каналов — настройка быстрее ручного ввода RTMP: выбрал Streamate, вставил ключ, готово.

Гайд по настройке
Коротко
Вещание на Streamate через SplitCam: установите SplitCam, соберите сцену, на Streamate через SM Connect → Start Show скопируйте ключ, затем в SplitCam откройте Stream Settings → Add Channel → Streamate и вставьте.

Видео-гайд

Пошагово

1
Скачайте и установите SplitCam

SplitCam — бесплатная программа для трансляций на Windows и macOS. Скачайте и запустите установщик — без регистрации, без водяного знака, без ограничения по времени. Именно SplitCam отправляет видео на Streamate.

2
Соберите сцену

Добавьте веб-камеру, накладки и фильтры в SplitCam до получения ключа.

3
Получите ключ трансляции Streamate

На Streamate откройте SM Connect, примите условия, нажмите Start Show слева и закройте открывшееся окно — затем скопируйте ключ трансляции. В SplitCam откройте Stream Settings → Add Channel, выберите Streamate из списка и вставьте ключ. Зелёный слайдер подтверждает подключение.

4
Добавьте Streamate как канал

В SplitCam откройте Stream Settings → Add Channel, выберите Streamate из встроенного списка и вставьте ключ из SM Connect — ручной RTMP-URL не нужен. Зафиксируйте разрешение на 1080p.

5
Нажмите Go Live

Нажмите Go Live в SplitCam — зелёный слайдер подтвердит подключение. Поток адаптивный, поэтому пониженный битрейт на статичной картинке — это норма.

Советы

Streamate — встроенный канал

Ручной RTMP-URL не нужен — Streamate есть в списке Add Channel SplitCam, просто выберите его и вставьте ключ.

Закройте окно SM Connect

После Start Show Streamate открывает окно — закройте его; оно нужно было только чтобы показать ключ.

Зафиксируйте разрешение на 1080p

Поле Video Resolution в SM Connect может само переключиться на 1440 — оно реально не передаётся и тихо роняет качество, поэтому выставьте 1080p вручную. Падение битрейта на статичной картинке — это нормально: поток Streamate адаптивный.

Проводное подключение

Поток Streamate и так снижает битрейт под нагрузкой — не добавляйте сверху нестабильность Wi-Fi, протяните Ethernet-кабель.

Сначала тест через SM Connect

Запустите короткое приватное шоу через SM Connect, чтобы проверить зелёный слайдер и сцену до выхода в публику.

Вопросы и ответы

Streamate встроен в SplitCam?

Да — Streamate есть в списке Add Channel SplitCam, вы выбираете его вместо ручного ввода RTMP-URL.

Где ключ трансляции Streamate?

SM Connect → принять условия → Start Show → закрыть окно → скопировать ключ.

SplitCam бесплатен для Streamate?

Да — бесплатно, без водяного знака и без лимита по времени; а так как Streamate — встроенный канал SplitCam, отдельной платы за кодировщик тоже нет.

Какой битрейт для Streamate?

Жёсткого потолка Streamate не задаёт — 3500–6000 Кбит/с для 1080p работает хорошо. Поток адаптивный, поэтому пониженный битрейт на статичной картинке — это норма, а не сбой.
