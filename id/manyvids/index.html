Cara siaran di MV Live dengan SplitCam

ManyVids adalah platform creator economy — penjualan klip, video custom, langganan fan club, dan produk live-streaming MV Live. Broadcaster Creator Studio default jalan di browser, tapi ia juga membuka jalur External Encoder standar yang dihubungkan oleh SplitCam gratis — memungkinkan kamu siaran dengan scene multi-kamera, overlay, dan filter di platform yang sama-sama ramah creator.

Jawaban singkat
Siaran di MV Live dengan SplitCam: pasang SplitCam, susun scene, di Creator Studio buka MV Live → Broadcast Settings → External Encoder, salin URL server dan stream key, tempel di SplitCam, Go Live.

Langkah demi langkah

1
Unduh dan pasang SplitCam

SplitCam adalah software live-streaming gratis untuk Windows dan macOS — tanpa daftar, tanpa kartu, tanpa watermark.

2
Susun scene-mu

Buka SplitCam dan tambahkan webcam. Tumpuk overlay, teks, kamera kedua atau HP-mu, beauty filter atau latar AI — pas untuk build goal-reveal MV Live dan reward trigger.

3
Ambil URL dan stream key MV Live

Masuk ke akun creator ManyVids, buka Creator Studio, ke MV Live → Broadcast Settings → External Encoder. Halaman menampilkan URL server dan stream key unik. Salin keduanya.

4
Hubungkan SplitCam ke MV Live

Di SplitCam buka Stream Settings, tempel URL server dan stream key MV Live ke kolom RTMP kustom. Atur bitrate 3.500–6.000 Kbps di 1920×1080, 30 fps, dengan keyframe 2 detik.

5
Klik Go Live

Tekan Go Live di SplitCam, lalu mulai siaran MV Live dari Creator Studio. Dalam ~10 detik stream-mu sampai ke audiens MV Live.

Tips pro

Creator economy, bukan sekadar live

ManyVids bukan platform cam murni — MV Live adalah satu jalur pendapatan di samping penjualan klip, video custom, dan langganan fan club. Pakai live untuk mendorong penonton ke jalur monetisasi lain.

Tipping token di dalam MV Live

MV Live punya sistem token tipping sendiri di dalam live room. Rancang goal menu dan reward trigger mirip Chaturbate / Stripchat — konversinya bagus di audiens ManyVids.

Browser vs encoder eksternal

Broadcaster browser bawaan Creator Studio single-kamera. SplitCam lewat External Encoder membuka scene multi-kamera, overlay, dan filter.

Pakai koneksi kabel

Ethernet mengalahkan Wi-Fi saat siaran panjang — frame yang hilang berarti tip yang hilang. Tarik kabel ke PC siaran.

Pemecahan masalah

Siaran ManyVids tersendat atau buffering

Hampir selalu bitrate diatur lebih tinggi dari yang mampu ditanggung upload-mu. Jalankan tes kecepatan bawaan SplitCam, lalu setel bitrate ke sekitar 75% dari upload terukur — 3.500–6.000 Kbps untuk 1080p, lebih rendah untuk 720p. Tersendat hilang begitu encoder berhenti melampaui koneksimu.

Frame hilang saat siaran ManyVids

Frame hilang berarti paket tidak sampai ke ManyVids tepat waktu — biasanya Wi-Fi tidak stabil. Pindah ke Ethernet kabel, tutup aplikasi pemakan bandwidth, dan turunkan bitrate sedikit. Satu lonjakan tidak masalah; kenaikan terus-menerus berarti koneksi tidak mengejar.

Layar hitam — penonton tidak melihat video di ManyVids

Kameramu tidak terpilih sebagai sumber aktif di SplitCam, atau aplikasi lain menahannya. Tutup Zoom, Skype, atau OBS, pilih ulang webcam di daftar sumber SplitCam, dan pastikan preview menampilkan gambarmu sebelum tekan Go Live.

ManyVids menolak stream key atau tidak terhubung

Salin ulang stream key — spasi di akhir atau kunci lama yang sudah di-reset adalah penyebab paling umum. Pastikan URL server cocok dengan yang ditampilkan ManyVids dan bahwa siaran encoder eksternal aktif di akunmu. Slider hijau di Stream Settings SplitCam menandakan kunci valid.

Tidak ada audio atau audio tidak sinkron di ManyVids

Pilih SplitCam sebagai kamera DAN mikrofon, dan pilih mikrofon aslimu di dalam sumber audio SplitCam. Jika audio tertinggal dari video, turunkan resolusi satu tingkat — encoder kelebihan beban dan audio menunggu frame yang terlambat.

FAQ

Apakah MV Live (ManyVids) mendukung encoder eksternal seperti SplitCam?

Ya — bagian MV Live di Creator Studio menyertakan opsi External Encoder di bawah Broadcast Settings. URL server RTMP standar dan stream key; OBS, SplitCam, vMix semua terhubung.

Di mana saya ambil stream key MV Live?

Creator Studio → MV Live → Broadcast Settings → External Encoder. URL server dan stream key muncul di sana — salin keduanya ke kolom RTMP kustom SplitCam.

Bitrate berapa untuk MV Live?

Dorong 1920×1080 di 30 fps, 3.500–6.000 Kbps dengan interval keyframe 2 detik. Jalankan speed test bawaan SplitCam dulu.

Apakah SplitCam gratis untuk MV Live?

Ya — SplitCam gratis, tanpa watermark, tanpa batas waktu. Opsi External Encoder ManyVids di Creator Studio juga gratis.
