Cara menambah MFC Alerts ke siaranmu

Langkah demi langkah

1
Pasang SplitCam dan tambah webcam

Pasang SplitCam gratis untuk Windows atau macOS, lalu tambahkan webcam sebagai sumber. MFC Alerts duduk sebagai lapisan di atas kamera ini.

2
Buka tab Browser dan ke mfcalerts.com

Di SplitCam buka tab Browser dan buka www.mfcalerts.com. Masuk, atau daftar jika belum punya akun mfcalerts.com.

3
Salin URL alert-mu

Di mfcalerts.com gunakan Copy to clipboard untuk menyalin URL alert pribadimu — inilah halaman yang merender animasi tip.

4
Tambah URL sebagai lapisan Browser — di atas webcam

Tempel URL ke jendela Browser SplitCam dan klik Add. Lalu atur ulang daftar sumber agar MFC Alerts di atas webcam (menu 3 titik → Move Up). Jika di bawah, efek tetap tersembunyi.

5
Uji dengan tip tes

Buka Settings → Send test tip dan pastikan efek alert muncul di atas kamera. Lalu siaran normal di MyFreeCams — tip asli kini memicu animasi.

Tips pro

MFC Alerts harus di atas webcam

Kesalahan paling umum: jika lapisan MFC Alerts di bawah webcam di daftar sumber, efek tetap tersembunyi. Pindahkan ke atas.

Perlu akun mfcalerts.com

URL alert bersifat pribadi — daftar di mfcalerts.com dulu jika belum punya akun.

Kirim tip tes sebelum live

Gunakan Settings → Send test tip untuk memastikan overlay bekerja — jangan menemukan kesalahan di tengah show.

Pakai koneksi kabel

Ethernet mengalahkan Wi-Fi saat siaran panjang — frame yang hilang berarti tip yang hilang. Tarik kabel ke PC siaran.

Pemecahan masalah

Siaran MFC Alerts tersendat atau buffering

Hampir selalu bitrate diatur lebih tinggi dari yang mampu ditanggung upload-mu. Jalankan tes kecepatan bawaan SplitCam, lalu setel bitrate ke sekitar 75% dari upload terukur — 3.500–6.000 Kbps untuk 1080p, lebih rendah untuk 720p. Tersendat hilang begitu encoder berhenti melampaui koneksimu.

Frame hilang saat siaran MFC Alerts

Frame hilang berarti paket tidak sampai ke MFC Alerts tepat waktu — biasanya Wi-Fi tidak stabil. Pindah ke Ethernet kabel, tutup aplikasi pemakan bandwidth, dan turunkan bitrate sedikit. Satu lonjakan tidak masalah; kenaikan terus-menerus berarti koneksi tidak mengejar.

Layar hitam — penonton tidak melihat video di MFC Alerts

Kameramu tidak terpilih sebagai sumber aktif di SplitCam, atau aplikasi lain menahannya. Tutup Zoom, Skype, atau OBS, pilih ulang webcam di daftar sumber SplitCam, dan pastikan preview menampilkan gambarmu sebelum tekan Go Live.

MFC Alerts menolak stream key atau tidak terhubung

Salin ulang stream key — spasi di akhir atau kunci lama yang sudah di-reset adalah penyebab paling umum. Pastikan URL server cocok dengan yang ditampilkan MFC Alerts dan bahwa siaran encoder eksternal aktif di akunmu. Slider hijau di Stream Settings SplitCam menandakan kunci valid.

Tidak ada audio atau audio tidak sinkron di MFC Alerts

Pilih SplitCam sebagai kamera DAN mikrofon, dan pilih mikrofon aslimu di dalam sumber audio SplitCam. Jika audio tertinggal dari video, turunkan resolusi satu tingkat — encoder kelebihan beban dan audio menunggu frame yang terlambat.

FAQ

Apa itu MFC Alerts?

Sistem notifikasi untuk MyFreeCams yang menampilkan efek video di siaranmu saat penonton mengirim tip — ditambahkan sebagai overlay Browser di SplitCam.

Kenapa MFC Alerts saya tidak muncul?

Apakah perlu akun untuk MFC Alerts?

Ya — daftar di mfcalerts.com untuk mendapat URL alert pribadimu.

Apakah SplitCam gratis untuk ini?

Ya — SplitCam gratis, tanpa watermark dan tanpa batas waktu, dan overlay browser MFC Alerts berjalan di dalamnya tanpa biaya tambahan.

Berapa penghasilan model di MFC Alerts?

Penghasilan di MFC Alerts bergantung pada ukuran audiens, jam siaran, dan kebiasaan memberi tip. Broadcaster aktif biasanya membawa pulang $200–$3.000 per bulan; performer top mencapai $10.000+. Bagian pendapatan Anda mengikuti struktur komisi MFC Alerts — periksa perjanjian model sebelum mulai siaran.

Apakah MFC Alerts aman untuk broadcaster?

MFC Alerts mewajibkan verifikasi usia dan identitas sebelum pembayaran, yang melindungi model dari penipuan. Gunakan nama panggung, jangan pernah membagikan data pribadi di kamera, aktifkan geo-block untuk menyembunyikan siaran dari wilayah asal Anda, dan perlakukan setiap permintaan penonton sebagai transaksi. Overlay dan AI background SplitCam juga dapat menyembunyikan atau mengganti lingkungan asli Anda.

Dokumen apa yang dibutuhkan untuk menjadi model di MFC Alerts?

MFC Alerts biasanya membutuhkan kartu identitas berfoto yang dikeluarkan pemerintah (paspor, SIM, atau KTP), selfie sambil memegang ID, dan formulir pajak/pembayaran (W-9 untuk AS, W-8BEN untuk non-AS). Persetujuan biasanya memakan waktu 24–72 jam; setelah disetujui Anda bisa langsung siaran di hari yang sama.

Bisakah saya siaran di MFC Alerts dari ponsel?

MFC Alerts biasanya menawarkan aplikasi broadcaster mobile atau broadcaster mobile-web, tetapi pengalamannya terbatas — tanpa overlay, tanpa kamera kedua, tanpa AI background. Untuk kualitas produksi penuh, siaran dari komputer dengan SplitCam dan gunakan ponsel Anda sebagai kamera kedua (SplitCam menerima input IP-camera dari ponsel).
